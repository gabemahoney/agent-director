# overwrite-3: --overwrite creates the file when the templates dir is empty

Maps to PRD AC 3. The result envelope must be opaque on did-something
vs no-op: `--overwrite` against a non-existent target must succeed and
create the file, with the same exit/stdout shape as the
overwrite-existing case. The Go layer pins envelope opacity; this case
pins the CLI-layer file-creation behaviour.

## Setup

Per-case `HOME=$(mktemp -d)` — templates dir does not exist yet.

## Steps

1. Confirm `$HOME/.agent-director/templates/` does NOT exist (per-case mktemp guarantees this).
2. Run `agent-director make-template --name fresh --overwrite --cwd /tmp/x` — must succeed.
3. Assert `$HOME/.agent-director/templates/fresh.toml` exists and is non-empty.
4. Assert the TOML body's `cwd` field contains `/tmp/x`.

## Pass criteria

- `make-template --overwrite` against an empty templates dir exits 0.
- The target file is created with non-zero size.
- The encoded TOML reflects the call's params.

## Script

```bash
set -euo pipefail
export HOME=$(mktemp -d)
TEMPLATE="$HOME/.agent-director/templates/fresh.toml"

# Sanity: per-case mktemp guarantees an empty $HOME with no agent-director state.
if [ -e "$HOME/.agent-director/templates" ]; then
    echo "FAIL: templates dir already exists in a freshly mktemp'd HOME ($HOME)"
    exit 1
fi

/usr/local/bin/agent-director make-template --name fresh --overwrite --cwd /tmp/x >/dev/null

if [ ! -f "$TEMPLATE" ]; then
    echo "FAIL: $TEMPLATE was not created"; exit 1
fi
size=$(stat -c '%s' "$TEMPLATE")
if [ "$size" -le 0 ]; then
    echo "FAIL: $TEMPLATE has size $size (zero-byte file)"; exit 1
fi
if ! grep -q 'cwd = "/tmp/x"' "$TEMPLATE"; then
    echo "FAIL: TOML body does not contain cwd=/tmp/x"; cat "$TEMPLATE"; exit 1
fi

echo "overwrite-3 ok: --overwrite against empty templates dir created the file"
```

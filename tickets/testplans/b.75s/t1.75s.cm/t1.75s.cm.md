# CI matrix (`ci-matrix`) testplan

t1 collector for Epic 6 of bee `b.qe2` (CI matrix ‚Äî
`.github/workflows/cabi-matrix.yml` plus
`docs/self-hosted-runner-setup.md`). Each t2 is a plain-shell case the
Docker harness executes. Runs under
`make test-docker EPIC=ci-matrix`; the driver
(`test/driver/run-testplan.sh`) resolves the slug by case-insensitive
substring match against this t1's frontmatter title, so the canonical
substring `ci-matrix` must remain in the title verbatim.

The cases pin the *shape and coverage* of the Epic's static
deliverables (the workflow YAML and the runbook). End-to-end "all
three matrix legs green on the Actions UI" verification is operator
action (push of the branch + self-hosted runner online) and is not
exercised by this testplan; those acceptance criteria are covered by
the cabi-matrix.yml run itself when the operator pushes.

## Driver mode

All cases use `DRIVER_MODE=shell` ‚Äî the default for
`make test-docker`. Each t2 body has a single fenced ` ```bash ` block
the driver executes after the per-case DB-reset fixture.

None of the cases here need API auth, a real `agent-director` binary,
a real Claude session, or the network beyond what `go install` needs
for case 1 (actionlint). The repo's read-only source bind-mount at
`/work/source` is sufficient for cases 2‚Äì5 (they are pure grep / awk
over committed files).

## Notes for the orchestrator

- Audit standard (SRD ¬ß17): run
  `make test-docker EPIC=ci-matrix` yourself, inspect the JSON stream
  case-by-case, confirm each `t2` emitted `{"status":"pass"}`.
- A contrived regression demonstrates the testplan has teeth: e.g.
  changing `ubuntu-22.04` to `ubuntu-latest` should fail
  `ci-matrix-3-no-floating-tags`; deleting the `## 4. Security`
  heading from the runbook should fail `ci-matrix-4-runbook-seven-sections`;
  removing the `Fork pull request workflows` phrase should fail
  `ci-matrix-5-runbook-public-repo-security`.
- The cases do not launch any tmux sessions, so they leave no `cd-*`
  sessions for the DB-reset fixture to sweep.

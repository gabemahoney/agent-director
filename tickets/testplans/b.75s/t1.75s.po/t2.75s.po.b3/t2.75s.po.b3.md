# bootstrap-3-newer-left-alone

Pins SR-1.4 newer branch and SR-1.6 never-suppress: when the destination's
frontmatter `version:` is strictly greater than the source per semver §11,
the postinstall leaves the destination untouched and emits a single
never-suppressed warning line to stderr. Invoke the postinstall directly
(see b2 notes on why `bun add` cannot expose lifecycle output).

## Pass criteria

- Postinstall exits 0.
- Destination tree-hash is byte-identical between pre- and post-invocation.
- Exactly one `agent-director: ` line emitted; it contains the literal
  substring `is version 9.9.9 (newer than package's `.

## Script

```bash
set -euo pipefail

FIXDIR=/tmp/postinstall-skill-copy-fixtures
if [ ! -f "$FIXDIR/.ready" ]; then
    echo "FAIL: fixture cache not populated; bootstrap-1 must run first"
    exit 1
fi

TESTHOME=$(mktemp -d /tmp/po-b3-home.XXXXXX)
PROJ=$(mktemp -d /tmp/po-b3-proj.XXXXXX)

cat > "$PROJ/package.json" <<JSON
{
  "name": "consumer-b3",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

(cd "$PROJ" && HOME="$TESTHOME" bun add --ignore-scripts \
    "file:$FIXDIR/agent-director.tgz" >/dev/null 2>&1)

PI="$PROJ/node_modules/agent-director/scripts/postinstall.ts"
SRC="$PROJ/node_modules/agent-director/skills/install-agent-director"
DEST="$TESTHOME/.claude/skills/install-agent-director"

# Pre-seed destination then mutate its frontmatter to 9.9.9.
mkdir -p "$DEST"
cp "$SRC/SKILL.md" "$SRC/install.sh" "$SRC/uninstall.sh" "$DEST/"
chmod 0755 "$DEST/install.sh" "$DEST/uninstall.sh"
sed -i 's/^version: .*/version: 9.9.9/' "$DEST/SKILL.md"

tree_hash() {
    (cd "$1" && find . -type f -print0 | sort -z | xargs -0 sha256sum | sha256sum | awk '{print $1}')
}
HASH_BEFORE=$(tree_hash "$DEST")

LOG=$(mktemp /tmp/po-b3-log.XXXXXX)
EXIT=0
HOME="$TESTHOME" bun run "$PI" > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: postinstall exit=$EXIT"; sed 's/^/  /' "$LOG"; exit 1
fi

HASH_AFTER=$(tree_hash "$DEST")
if [ "$HASH_BEFORE" != "$HASH_AFTER" ]; then
    echo "FAIL: destination tree changed; before=$HASH_BEFORE after=$HASH_AFTER"
    exit 1
fi

POSTINSTALL_LINES=$(grep -c '^agent-director: ' "$LOG" || true)
if [ "$POSTINSTALL_LINES" -ne 1 ]; then
    echo "FAIL: newer branch emitted $POSTINSTALL_LINES line(s) (want 1)"
    grep '^agent-director: ' "$LOG" | sed 's/^/  /'
    exit 1
fi
if ! grep -q "is version 9.9.9 (newer than package's " "$LOG"; then
    echo "FAIL: newer-branch warning not found"
    sed 's/^/  /' "$LOG"
    exit 1
fi

rm -rf "$TESTHOME" "$PROJ" "$LOG"
echo "bootstrap-3-newer-left-alone ok: dest unchanged; single SR-1.4 warning emitted"
```

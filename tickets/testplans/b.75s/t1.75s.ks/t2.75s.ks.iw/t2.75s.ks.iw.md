# pause-1: pause on an ended row is a no-op success

Per SRD §9 + Epic 5 task ticket: terminal states are no-op success
for pause. No `/exit` is sent; the verb returns 0.

## Setup

Per-case DB-reset. `ended` row; no tmux session needed.

## Steps

1. Seed `spawns` row at state `ended`, session_name `cd-p1`.
2. `agent-director pause --claude-instance-id id-p1`.

## Pass criteria

- pause exits 0.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.agent-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-p1', 'ended', '/tmp', 'cd-p1', 'off');"
agent-director pause --claude-instance-id id-p1 >/dev/null
echo "pause-1 ok: ended row → pause → exit 0 (no-op)"
```

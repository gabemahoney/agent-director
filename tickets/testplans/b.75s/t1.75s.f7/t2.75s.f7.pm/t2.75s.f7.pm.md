# install-7: uninstall --purge --force removes everything

Per SRD §16.2: --purge is the explicit nuke path.

## Setup

Per-case DB-reset.

## Steps

1. install.
2. uninstall --purge --force.
3. Verify ~/.agent-director/ is gone entirely.

## Pass criteria

- After uninstall --purge --force: ~/.agent-director/ does not exist.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)

HOME="$TESTHOME" /opt/skills/install-agent-director/install.sh \
    --binary /usr/local/bin/agent-director --no-symlink >/dev/null
HOME="$TESTHOME" /opt/skills/install-agent-director/uninstall.sh --purge --force >/dev/null

if [ -d "$TESTHOME/.agent-director" ]; then
    echo "FAIL: ~/.agent-director still exists after --purge --force"; exit 1
fi
echo "install-7 ok: --purge --force removed ~/.agent-director entirely"
```

# recov-6: delete returns per-row results; batch does not abort on a bad id

Maps to Epic 8 AC #3. A mix of valid and bogus ids returns a map
recording each outcome; the valid rows are deleted while the bogus
one surfaces `ErrSpawnNotFound`.

## Setup

Per-case DB-reset.

## Steps

1. INSERT two rows (`id-d-1`, `id-d-2`).
2. `delete --claude-instance-id id-d-1 --claude-instance-id bogus --claude-instance-id id-d-2`.
3. Expect result map `{id-d-1:ok, bogus:ErrSpawnNotFound, id-d-2:ok}`.
4. Both valid rows are gone; bogus was never inserted.

## Pass criteria

- delete exits 0.
- result.results map matches expectations.
- Valid rows removed from DB.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.agent-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-d-1', 'ended', '/tmp', 'cd-d-1', 'off'),
          ('id-d-2', 'ended', '/tmp', 'cd-d-2', 'off');"

res=$(agent-director delete --claude-instance-id id-d-1 \
        --claude-instance-id bogus --claude-instance-id id-d-2)
ok1=$(printf '%s' "$res" | jq -r '.results["id-d-1"]')
ok2=$(printf '%s' "$res" | jq -r '.results["id-d-2"]')
b=$(printf '%s' "$res" | jq -r '.results.bogus')
if [ "$ok1" != "ok" ] || [ "$ok2" != "ok" ] || [ "$b" != "ErrSpawnNotFound" ]; then
    echo "FAIL: results=$res; want id-d-1:ok id-d-2:ok bogus:ErrSpawnNotFound"; exit 1
fi

n=$(sqlite3 "$HOME/.agent-director/state.db" \
    "SELECT COUNT(*) FROM spawns WHERE claude_instance_id IN ('id-d-1','id-d-2','bogus');")
if [ "$n" != "0" ]; then
    echo "FAIL: $n row(s) remain; want 0 (both valids deleted)"; exit 1
fi
echo "recov-6 ok: per-row result map; partial failure didn't abort the batch"
```

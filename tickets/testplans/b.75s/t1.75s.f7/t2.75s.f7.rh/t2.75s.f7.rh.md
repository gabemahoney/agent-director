# install-8: upgrade swaps the canonical symlink to a new versioned binary

Per SRD §16.2: the upgrade-safety pattern writes the new binary at a
versioned path AND atomically swaps the canonical symlink. The
previous versioned file stays in place for rollback.

## Setup

Per-case DB-reset.

## Steps

1. First install with --version v1-old.
2. Note the symlink's target (should be claude-director.v1-old).
3. Second install with --version v2-new.
4. Confirm the symlink now points at claude-director.v2-new.
5. Confirm the old versioned file (claude-director.v1-old) is STILL
   present (retained for rollback).

## Pass criteria

- First-install symlink target is claude-director.v1-old.
- Second-install symlink target is claude-director.v2-new.
- Both versioned files exist after second install.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)

HOME="$TESTHOME" /opt/skills/install-claude-director/install.sh \
    --binary /usr/local/bin/claude-director --no-symlink --version v1-old >/dev/null

canonical="$TESTHOME/.claude-director/bin/claude-director"
target1=$(readlink "$canonical")
case "$target1" in
    *claude-director.v1-old) ;;
    *) echo "FAIL: first install symlink target=$target1; want *.v1-old"; exit 1 ;;
esac

HOME="$TESTHOME" /opt/skills/install-claude-director/install.sh \
    --binary /usr/local/bin/claude-director --no-symlink --version v2-new >/dev/null

target2=$(readlink "$canonical")
case "$target2" in
    *claude-director.v2-new) ;;
    *) echo "FAIL: second install symlink target=$target2; want *.v2-new"; exit 1 ;;
esac

# Old versioned file retained for rollback.
if [ ! -f "$TESTHOME/.claude-director/bin/claude-director.v1-old" ]; then
    echo "FAIL: previous versioned file claude-director.v1-old missing — rollback path broken"
    ls -la "$TESTHOME/.claude-director/bin/"
    exit 1
fi
echo "install-8 ok: symlink swapped; v1-old retained for rollback"
```

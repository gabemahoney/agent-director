# bootstrap-4-older-overwritten

Pins SR-1.4 older sub-case and SR-1.5 atomic-write+backup: when the
destination's frontmatter `version:` is strictly less than the source per
semver §11, the postinstall overwrites the destination AND writes a
timestamped backup sibling under
`${HOME}/.claude/skills/install-agent-director.bak.<unix-ts>` holding the
prior tree. Invoke the postinstall directly (see b2 notes on why `bun add`
cannot expose lifecycle output).

## Pass criteria

- Postinstall exits 0.
- Destination tree-hash equals the source tree-hash after invocation.
- A `${HOME}/.claude/skills/install-agent-director.bak.<unix-ts>` sibling
  exists with frontmatter `version: 0.0.0` (the staged prior state).
- Combined stdout+stderr `agent-director: ` line count is ≤ 5 (older
  branch is silent on the default-quiet success path).

## Script

```bash
set -euo pipefail

FIXDIR=/tmp/postinstall-skill-copy-fixtures
if [ ! -f "$FIXDIR/.ready" ]; then
    echo "FAIL: fixture cache not populated; bootstrap-1 must run first"
    exit 1
fi

TESTHOME=$(mktemp -d /tmp/po-b4-home.XXXXXX)
PROJ=$(mktemp -d /tmp/po-b4-proj.XXXXXX)

cat > "$PROJ/package.json" <<JSON
{
  "name": "consumer-b4",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

(cd "$PROJ" && HOME="$TESTHOME" bun add --ignore-scripts \
    "file:$FIXDIR/agent-director.tgz" >/dev/null 2>&1)

PI="$PROJ/node_modules/agent-director/scripts/postinstall.ts"
SRC="$PROJ/node_modules/agent-director/skills/install-agent-director"
DEST="$TESTHOME/.claude/skills/install-agent-director"

# Pre-seed destination then downgrade its frontmatter to 0.0.0.
mkdir -p "$DEST"
cp "$SRC/SKILL.md" "$SRC/install.sh" "$SRC/uninstall.sh" "$DEST/"
chmod 0755 "$DEST/install.sh" "$DEST/uninstall.sh"
sed -i 's/^version: .*/version: 0.0.0/' "$DEST/SKILL.md"

tree_hash() {
    (cd "$1" && find . -type f -print0 | sort -z | xargs -0 sha256sum | sha256sum | awk '{print $1}')
}
HASH_SOURCE=$(tree_hash "$SRC")

LOG=$(mktemp /tmp/po-b4-log.XXXXXX)
EXIT=0
HOME="$TESTHOME" bun run "$PI" > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: postinstall exit=$EXIT"; sed 's/^/  /' "$LOG"; exit 1
fi

HASH_DEST=$(tree_hash "$DEST")
if [ "$HASH_SOURCE" != "$HASH_DEST" ]; then
    echo "FAIL: destination did not match source; source=$HASH_SOURCE dest=$HASH_DEST"
    exit 1
fi

SKILLS_DIR="$TESTHOME/.claude/skills"
BAK=$(ls -d "$SKILLS_DIR"/install-agent-director.bak.* 2>/dev/null | head -1 || true)
if [ -z "$BAK" ]; then
    echo "FAIL: no install-agent-director.bak.<ts> sibling under $SKILLS_DIR"
    ls "$SKILLS_DIR" | sed 's/^/  /'
    exit 1
fi

BAK_V=$(awk '/^version:/ {print $2; exit}' "$BAK/SKILL.md")
if [ "$BAK_V" != "0.0.0" ]; then
    echo "FAIL: backup carries version=$BAK_V (want 0.0.0 — the prior staged state)"
    exit 1
fi

POSTINSTALL_LINES=$(grep -c '^agent-director: ' "$LOG" || true)
if [ "$POSTINSTALL_LINES" -gt 5 ]; then
    echo "FAIL: older branch emitted $POSTINSTALL_LINES lines (cap is 5)"
    grep '^agent-director: ' "$LOG" | sed 's/^/  /'
    exit 1
fi

rm -rf "$TESTHOME" "$PROJ" "$LOG"
echo "bootstrap-4-older-overwritten ok: dest==source; backup at $(basename "$BAK") holds version=0.0.0; postinstall lines=$POSTINSTALL_LINES"
```

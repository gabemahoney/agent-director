# resume-7: parent_id is re-derived on every resume

Per SRD §7.5: `parent_id` records who currently owns the Spawn, not
who originally created it. spawn B inside A → B's parent is A.
Resume B from a bare shell → B's parent goes NULL.

## Setup

Per-case DB-reset.

## Steps

1. Spawn A (no parent env).
2. Spawn B with `AGENT_DIRECTOR_INSTANCE_ID=<A>` in env → B's
   parent_id is A.
3. Synth SessionStart for B → claude_session_id populated.
4. Seed JSONL on disk.
5. Synth SessionEnd for B → state=ended.
6. Free B's tmux session via tmux kill-session.
7. `resume B` from a bare shell (env unset) → B's parent_id is now
   NULL.

## Pass criteria

- B's parent_id before resume is A.
- B's parent_id after resume is empty (NULL in DB).

## Script

```bash
set -euo pipefail
a=$(agent-director spawn --cwd /tmp | jq -r '.claude_instance_id')
test -n "$a"

b=$(AGENT_DIRECTOR_INSTANCE_ID="$a" agent-director spawn --cwd /tmp | jq -r '.claude_instance_id')
test -n "$b"

# Sanity check: B's parent is A.
parent=$(agent-director get --claude-instance-id "$b" | jq -r '.parent_id')
if [ "$parent" != "$a" ]; then
    echo "FAIL: precondition parent_id=$parent; want $a"; exit 1
fi

sess_b=$(agent-director get --claude-instance-id "$b" | jq -r '.tmux_session_name')
session_id="r7-session-uuid"
printf '%s' "{\"hook_event_name\":\"SessionStart\",\"transcript_path\":\"/x/${session_id}.jsonl\"}" \
    | AGENT_DIRECTOR_INSTANCE_ID="$b" agent-director hook >/dev/null

mkdir -p "$HOME/.claude/projects/-tmp"
echo '{}' > "$HOME/.claude/projects/-tmp/${session_id}.jsonl"

printf '%s' '{"hook_event_name":"SessionEnd"}' \
    | AGENT_DIRECTOR_INSTANCE_ID="$b" agent-director hook >/dev/null

tmux kill-session -t "$sess_b" 2>/dev/null || true

# Bare-shell resume — explicitly unset the env var.
unset AGENT_DIRECTOR_INSTANCE_ID
agent-director resume --claude-instance-id "$b" >/dev/null

# B's parent is now NULL (empty string in the JSON envelope).
parent=$(agent-director get --claude-instance-id "$b" | jq -r '.parent_id // empty')
if [ -n "$parent" ]; then
    echo "FAIL: parent_id=$parent after bare-shell resume; want empty"; exit 1
fi
echo "resume-7 ok: bare-shell resume nulled parent_id (was $a)"
```

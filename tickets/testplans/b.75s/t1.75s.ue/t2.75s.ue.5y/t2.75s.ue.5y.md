# resume-8: ended_at is cleared on resurrection

Per SRD §8.1 + the store contract: when SessionStart on a resurrected
Claude fires, the row transitions to `waiting` AND `ended_at` is
cleared. This is the store-side half of the resume contract.

## Setup

Per-case DB-reset.

## Steps

1. Spawn → synth SessionStart → seed JSONL → synth SessionEnd. Row
   state is `ended` with ended_at populated.
2. Verify ended_at is non-null (precondition).
3. Free tmux session.
4. `resume`.
5. Synth a fresh SessionStart for the resumed Claude (the actual
   claude --resume process wouldn't drive this in the harness).
6. Row state is `waiting` AND ended_at is NULL.

## Pass criteria

- After SessionEnd, ended_at is non-null (precondition).
- After resume + simulated SessionStart, ended_at is NULL.
- Row state is `waiting`.

## Script

```bash
set -euo pipefail
id=$(agent-director spawn --cwd /tmp | jq -r '.claude_instance_id')
sess=$(agent-director get --claude-instance-id "$id" | jq -r '.tmux_session_name')

session_id="r8-session-uuid"
printf '%s' "{\"hook_event_name\":\"SessionStart\",\"transcript_path\":\"/x/${session_id}.jsonl\"}" \
    | AGENT_DIRECTOR_INSTANCE_ID="$id" agent-director hook >/dev/null

mkdir -p "$HOME/.claude/projects/-tmp"
echo '{}' > "$HOME/.claude/projects/-tmp/${session_id}.jsonl"

printf '%s' '{"hook_event_name":"SessionEnd","reason":"logout"}' \
    | AGENT_DIRECTOR_INSTANCE_ID="$id" agent-director hook >/dev/null

ended_before=$(sqlite3 "$HOME/.agent-director/state.db" \
    "SELECT ended_at FROM spawns WHERE claude_instance_id='$id';")
if [ -z "$ended_before" ]; then
    echo "FAIL: precondition ended_at empty after SessionEnd"; exit 1
fi

tmux kill-session -t "$sess" 2>/dev/null || true
agent-director resume --claude-instance-id "$id" >/dev/null

# Synth a fresh SessionStart for the resumed claude. New session uuid
# so the rotation behavior is also exercised.
new_session_id="r8-session-uuid-resumed"
mkdir -p "$HOME/.claude/projects/-tmp"
echo '{}' > "$HOME/.claude/projects/-tmp/${new_session_id}.jsonl"
printf '%s' "{\"hook_event_name\":\"SessionStart\",\"transcript_path\":\"/x/${new_session_id}.jsonl\"}" \
    | AGENT_DIRECTOR_INSTANCE_ID="$id" agent-director hook >/dev/null

state=$(agent-director get --claude-instance-id "$id" | jq -r '.state')
if [ "$state" != "waiting" ]; then
    echo "FAIL: state=$state; want waiting"; exit 1
fi
ended_after=$(sqlite3 "$HOME/.agent-director/state.db" \
    "SELECT IFNULL(ended_at,'') FROM spawns WHERE claude_instance_id='$id';")
if [ -n "$ended_after" ]; then
    echo "FAIL: ended_at=$ended_after; want empty (cleared on resurrection)"; exit 1
fi
echo "resume-8 ok: resurrection cleared ended_at; state=waiting"
```

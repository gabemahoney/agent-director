# spawn-10: `--tmux-session-name` flag end-to-end

Pins Epic 1 (`t1.6ya.6r`) Acceptance Criteria items 2–8 in one shell
script: happy path (verbatim persisted name), validation failures
(reserved char, control char, > 64 bytes, empty), live-collision
(tmux's own refusal, no `ErrTmuxSessionNameTaken` sentinel), and
name-reuse-after-end. The `list --tmux-session-name` filter case is
Epic 2 (`t1.6ya.pg`) and is out of scope here.

## Setup

Per-case DB-reset has wiped `~/.agent-director/state.db` and killed
any `cd-*` tmux sessions. The container has `agent-director`,
`tmux`, `sqlite3`, `jq` on PATH. The case uses the bespoke session
name `e1-happy` which does **not** match the harness's `cd-*` sweep
prefix, so the script explicitly cleans it up in a trap.

## Steps

1. Happy path: `spawn --tmux-session-name e1-happy`; assert `get`
   returns `tmux_session_name == "e1-happy"` verbatim.
2. Validation: `--tmux-session-name 'bad:name'` → exit 1 + stderr
   JSON `err_name == "ErrTmuxSessionNameInvalid"`.
3. Validation: `--tmux-session-name $'bad\x01name'` → exit 1 +
   `ErrTmuxSessionNameInvalid`.
4. Validation: 65-byte all-`a` name → exit 1 +
   `ErrTmuxSessionNameTooLong`.
5. Validation: `--tmux-session-name=` (explicit empty) → exit 1 +
   `ErrTmuxSessionNameEmpty`.
6. Live-collision: while step 1's tmux session is alive, repeat
   `--tmux-session-name e1-happy` → exit 1; stderr does NOT contain
   `ErrTmuxSessionNameTaken` (the wrapped tmux error surfaces).
7. Name reuse after end: `agent-director kill` step 1's spawn, then
   re-spawn with `--tmux-session-name e1-happy` → exit 0.

## Pass criteria

- Step 1: exit 0, `get` JSON `tmux_session_name == "e1-happy"`.
- Steps 2–5: exit 1, last JSON line on stderr has the matching
  `err_name`.
- Step 6: exit 1, stderr does not contain the literal string
  `ErrTmuxSessionNameTaken`.
- Step 7: exit 0, `get` confirms the new spawn carries the reused
  name.

## Script

```bash
set -euo pipefail

# tmux sessions named "e1-happy" sit outside the harness's cd-* sweep;
# clean up on exit regardless of step outcome.
trap 'tmux kill-session -t e1-happy 2>/dev/null || true' EXIT

# err_name extracts the err_name field from a stderr blob, tolerating
# the optional pre-trust soft-warning line that precedes the JSON
# envelope.
err_name() {
    printf '%s\n' "$1" | awk '/^\{/{ln=$0} END{print ln}' | jq -r '.err_name'
}

# Step 1: happy path
id1="$(agent-director spawn --cwd /tmp --tmux-session-name e1-happy \
    | jq -r '.claude_instance_id')"
test -n "$id1"
name1="$(agent-director get --claude-instance-id "$id1" | jq -r '.tmux_session_name')"
test "$name1" = "e1-happy"

# Step 2: reserved char
set +e
err="$(agent-director spawn --cwd /tmp --tmux-session-name 'bad:name' 2>&1 >/dev/null)"
code=$?
set -e
test "$code" -ne 0
test "$(err_name "$err")" = "ErrTmuxSessionNameInvalid"

# Step 3: control char
set +e
err="$(agent-director spawn --cwd /tmp --tmux-session-name $'bad\x01name' 2>&1 >/dev/null)"
code=$?
set -e
test "$code" -ne 0
test "$(err_name "$err")" = "ErrTmuxSessionNameInvalid"

# Step 4: too long
long="$(printf 'a%.0s' {1..65})"
set +e
err="$(agent-director spawn --cwd /tmp --tmux-session-name "$long" 2>&1 >/dev/null)"
code=$?
set -e
test "$code" -ne 0
test "$(err_name "$err")" = "ErrTmuxSessionNameTooLong"

# Step 5: explicit empty
set +e
err="$(agent-director spawn --cwd /tmp --tmux-session-name= 2>&1 >/dev/null)"
code=$?
set -e
test "$code" -ne 0
test "$(err_name "$err")" = "ErrTmuxSessionNameEmpty"

# Step 6: live collision — tmux's own refusal, NO ErrTmuxSessionNameTaken.
set +e
err="$(agent-director spawn --cwd /tmp --tmux-session-name e1-happy 2>&1 >/dev/null)"
code=$?
set -e
test "$code" -ne 0
if printf '%s' "$err" | grep -q 'ErrTmuxSessionNameTaken'; then
    echo "FAIL: collision must not surface ErrTmuxSessionNameTaken; got: $err" >&2
    exit 1
fi

# Step 7: name reuse after end
agent-director kill --claude-instance-id "$id1" >/dev/null
# Belt-and-braces: kill the bare tmux session in case the row is in a
# state where the kill verb is a no-op (the row's tmux_session_name
# drives KillSession, so this is normally redundant).
tmux kill-session -t e1-happy 2>/dev/null || true
id2="$(agent-director spawn --cwd /tmp --tmux-session-name e1-happy \
    | jq -r '.claude_instance_id')"
test "$id2" != "$id1"
name2="$(agent-director get --claude-instance-id "$id2" | jq -r '.tmux_session_name')"
test "$name2" = "e1-happy"

echo "spawn-10 ok: id1=$id1 id2=$id2 (reused e1-happy)"
```

# cabi-3-unknown-handle-error

Pins Epic 2 AC: `ad_status` with a handle that was never minted returns an
`ErrUnknownHandle` error envelope — the error propagates cleanly as JSON with
`err_name == "ErrUnknownHandle"` and an `err_description` string.

## Setup

No DB state needed. The `.so` is reused from `/tmp/ad_cabi_test.so` if present;
otherwise built fresh. A C probe calls `ad_status` with a handle string that
was never registered in the C-ABI handle registry.

## Steps

1. Assert `gcc` is available in PATH; fail with a clear message if not.
2. Build `.so` to `/tmp/ad_cabi_test.so` if not already present.
3. Write C probe that `dlopen`s the `.so`, resolves `ad_status` and
   `ad_free_cstring`, calls `ad_status` with params containing a bogus handle
   and a `claude_instance_id`, prints the JSON result, and frees the string.
4. Compile with `gcc /tmp/probe3.c -ldl -o /tmp/probe3`.
5. Run `/tmp/probe3 "$SO"` and capture output.
6. Assert `err_name == "ErrUnknownHandle"` in the parsed response.

## Pass criteria

- `/tmp/probe3` exits 0.
- Response JSON has `err_name` field equal to `"ErrUnknownHandle"`.
- Script exits 0; final line matches `cabi-3 ok: ...`.

## Script

```bash
set -euo pipefail

if ! command -v gcc >/dev/null 2>&1; then
    echo "FAIL: gcc not found — add build-essential to test/Dockerfile"
    exit 1
fi

SO="/tmp/ad_cabi_test.so"
if [[ ! -f "$SO" ]]; then
    WORK=$(mktemp -d /tmp/cabi3_XXXXXX)
    cp -a /work/source/. "$WORK/"
    ( cd "$WORK" && mkdir -p dist && CGO_ENABLED=1 go build -buildmode=c-shared -o "$SO" ./pkg/cabi )
fi

cat > /tmp/probe3.c << 'CEOF'
#include <dlfcn.h>
#include <stdio.h>
#include <stdlib.h>
typedef char *(*ad_fn_t)(const char *);
typedef void (*free_fn_t)(char *);
int main(int argc, char **argv) {
    if (argc < 2) { fputs("usage: probe3 <so_path>\n", stderr); return 1; }
    void *lib = dlopen(argv[1], RTLD_NOW);
    if (!lib) { fprintf(stderr, "dlopen: %s\n", dlerror()); return 1; }
    ad_fn_t fn = (ad_fn_t)dlsym(lib, "ad_status");
    if (!fn) { fprintf(stderr, "dlsym ad_status: %s\n", dlerror()); return 1; }
    free_fn_t freefn = (free_fn_t)dlsym(lib, "ad_free_cstring");
    if (!freefn) { fprintf(stderr, "dlsym ad_free_cstring: %s\n", dlerror()); return 1; }
    char *result = fn(
        "{\"handle\":\"deadbeefdeadbeefdeadbeefdeadbeef\","
        "\"claude_instance_id\":\"x\"}"
    );
    printf("%s\n", result);
    freefn(result);
    dlclose(lib);
    return 0;
}
CEOF
gcc /tmp/probe3.c -ldl -o /tmp/probe3

OUTPUT=$(/tmp/probe3 "$SO")

ERR_NAME=$(printf '%s' "$OUTPUT" | jq -r '.err_name // empty')
if [[ "$ERR_NAME" != "ErrUnknownHandle" ]]; then
    echo "FAIL: expected err_name=ErrUnknownHandle, got response: $OUTPUT"
    exit 1
fi

echo "cabi-3 ok: ad_status unknown handle → err_name=$ERR_NAME"
```

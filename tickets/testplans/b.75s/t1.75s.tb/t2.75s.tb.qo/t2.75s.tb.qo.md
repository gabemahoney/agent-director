# ts-bun-3-envelope-diff

Pins Epic 5 AC: `make envelope-diff-ts` exits 0 — no envelope diffs between
the TS client library output and the CLI subprocess output for any callable
verb. The target runs `bun test test/envelope-diff.test.ts
test/envelope-diff-invariants.test.ts` after building `agent-director`,
`ts-helper`, and `fake-tmux` (T8 — envelope-diff regression suite). The TS
Client spawns the freshly-built `bin/agent-director` as a subprocess (no FFI).

## Setup

The script copies `/work/source` to a writable temp dir, runs
`bun install --no-progress`, then runs `make envelope-diff-ts` from the repo
root. The make target builds the Go binaries (agent-director, ts-helper,
fake-tmux) incrementally, then invokes bun test.

## Steps

1. Assert `bun` and `go` are in PATH; fail clearly if not.
2. Create workspace; `cp -a /work/source/.`.
3. `cd pkg/ts-bun-client && bun install --no-progress`.
4. `cd <workspace> && make envelope-diff-ts`; assert exit 0.

## Pass criteria

- `make envelope-diff-ts` exits 0.
- No envelope diff failures in the bun test output.
- Script exits 0; final line contains `ts-bun-3-envelope-diff ok:`.

## Script

```bash
set -euo pipefail

if ! command -v bun >/dev/null 2>&1; then
    echo "FAIL: bun not found in PATH — check test/Dockerfile BUN_VERSION ARG"
    exit 1
fi
if ! command -v go >/dev/null 2>&1; then
    echo "FAIL: go not found in PATH"
    exit 1
fi

export HOME=/tmp/bun-home-tsbun3
mkdir -p "$HOME"

WORK=$(mktemp -d /tmp/tsbun3_XXXXXX)
cp -a /work/source/. "$WORK/"
cd "$WORK"

# Install bun dependencies (needed by bun test inside make envelope-diff-ts).
# setup.ts (the bunfig preload) builds and stages the CLI binary into
# platforms/linux-x64/bin/ at test time.
cd "$WORK/pkg/ts-bun-client"
bun install --no-progress 2>/dev/null
cd "$WORK"

# Run the envelope-diff suite (builds agent-director, ts-helper, fake-tmux, then bun test)
OUT=$(mktemp /tmp/tsbun3_out.XXXXXX)
EXIT=0
make envelope-diff-ts > "$OUT" 2>&1 || EXIT=$?

if [[ "$EXIT" -ne 0 ]]; then
    echo "FAIL: make envelope-diff-ts exited with code $EXIT"
    tail -200 "$OUT" >&2
    exit 1
fi

echo "ts-bun-3-envelope-diff ok: make envelope-diff-ts exited 0"
```

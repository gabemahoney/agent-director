# spawn-4: relative cwd is rejected with ErrCwdNotAPath

Maps to Epic 3 AC #5 (first half): `spawn --cwd ./relative` →
`ErrCwdNotAPath`. The check fires before any tmux or DB side-effect,
so no row is created and no tmux session exists.

## Setup

Per-case DB-reset has wiped the DB. tmux has no `cd-*` sessions.

## Steps

1. `agent-director spawn --cwd ./relative` → expect non-zero exit.
2. Inspect stderr → JSON envelope `{"err_name":"ErrCwdNotAPath",...}`.
3. Inspect the spawns table → zero rows.
4. Inspect tmux → no `cd-*` sessions.

## Pass criteria

- Non-zero exit.
- err_name == ErrCwdNotAPath.
- Zero spawns rows.
- Zero matching tmux sessions.

## Script

```bash
set -euo pipefail
stderr_file=$(mktemp)
trap 'rm -f "$stderr_file"' EXIT
if agent-director spawn --cwd ./relative 2>"$stderr_file"; then
    echo "spawn unexpectedly succeeded"; exit 1
fi
err_name="$(jq -r '.err_name' <"$stderr_file")"
test "$err_name" = "ErrCwdNotAPath"
n="$(sqlite3 "$HOME/.agent-director/state.db" 'SELECT COUNT(*) FROM spawns;')"
test "$n" = "0"
if tmux ls 2>/dev/null | grep -qE '^cd-'; then
    echo "leftover cd-* tmux sessions present"; exit 1
fi
echo "spawn-4 ok: ErrCwdNotAPath; no row; no session"
```

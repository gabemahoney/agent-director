# cabi-2-ad-version-envelope

Pins Epic 2 AC: `ad_version` is handle-free and returns a well-formed success
envelope containing `version` and `commit` fields with no `err_name`. No
client handle is required or consulted.

## Setup

No DB state needed. The `.so` is reused from `/tmp/ad_cabi_test.so` if present
(built by cabi-1 in the same run); otherwise it is built fresh from
`/work/source`. A tiny C probe is written to `/tmp/probe2.c`, compiled with
`gcc -ldl`, and executed with the `.so` path as its sole argument.

## Steps

1. Assert `gcc` is available in PATH; fail with a clear message if not.
2. Build `.so` to `/tmp/ad_cabi_test.so` if not already present.
3. Write C probe that `dlopen`s the `.so`, resolves `ad_version` and
   `ad_free_cstring`, calls `ad_version("{}")`, prints the returned JSON, and
   frees the C string.
4. Compile with `gcc /tmp/probe2.c -ldl -o /tmp/probe2`.
5. Run `/tmp/probe2 "$SO"` and capture output.
6. Assert output is valid JSON with a non-empty `version` field.
7. Assert no `err_name` field is present in the response.

## Pass criteria

- `/tmp/probe2` exits 0.
- Response parses as JSON.
- `.version` field is non-empty.
- No `err_name` in the response.
- Script exits 0; final line matches `cabi-2 ok: ...`.

## Script

```bash
set -euo pipefail

if ! command -v gcc >/dev/null 2>&1; then
    echo "FAIL: gcc not found — add build-essential to test/Dockerfile"
    exit 1
fi

SO="/tmp/ad_cabi_test.so"
if [[ ! -f "$SO" ]]; then
    WORK=$(mktemp -d /tmp/cabi2_XXXXXX)
    cp -a /work/source/. "$WORK/"
    ( cd "$WORK" && mkdir -p dist && CGO_ENABLED=1 go build -buildmode=c-shared -o "$SO" ./pkg/cabi )
fi

cat > /tmp/probe2.c << 'CEOF'
#include <dlfcn.h>
#include <stdio.h>
#include <stdlib.h>
typedef char *(*ad_fn_t)(const char *);
typedef void (*free_fn_t)(char *);
int main(int argc, char **argv) {
    if (argc < 2) { fputs("usage: probe2 <so_path>\n", stderr); return 1; }
    void *lib = dlopen(argv[1], RTLD_NOW);
    if (!lib) { fprintf(stderr, "dlopen: %s\n", dlerror()); return 1; }
    ad_fn_t fn = (ad_fn_t)dlsym(lib, "ad_version");
    if (!fn) { fprintf(stderr, "dlsym ad_version: %s\n", dlerror()); return 1; }
    free_fn_t freefn = (free_fn_t)dlsym(lib, "ad_free_cstring");
    if (!freefn) { fprintf(stderr, "dlsym ad_free_cstring: %s\n", dlerror()); return 1; }
    char *result = fn("{}");
    printf("%s\n", result);
    freefn(result);
    dlclose(lib);
    return 0;
}
CEOF
gcc /tmp/probe2.c -ldl -o /tmp/probe2

OUTPUT=$(/tmp/probe2 "$SO")

if ! printf '%s' "$OUTPUT" | jq -e '.version' >/dev/null 2>&1; then
    echo "FAIL: no 'version' field in ad_version response: $OUTPUT"
    exit 1
fi

ERR=$(printf '%s' "$OUTPUT" | jq -r '.err_name // empty')
if [[ -n "$ERR" ]]; then
    echo "FAIL: unexpected err_name=$ERR in ad_version response: $OUTPUT"
    exit 1
fi

VERSION=$(printf '%s' "$OUTPUT" | jq -r '.version')
echo "cabi-2 ok: ad_version returned version=$VERSION"
```

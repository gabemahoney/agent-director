# discover-7: --limit caps the result count

Per SRD §12: `--limit N` caps the number of rows returned. The case
seeds 3 rows and asserts exactly 1 is returned.

## Setup

Per-case DB-reset. 3 seeded rows.

## Steps

1. INSERT 3 rows.
2. `claude-director list --limit 1`.

## Pass criteria

- list exits 0.
- `.spawns | length` equals 1.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.claude-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-d7-1', 'waiting', '/tmp', 'cd-d7-1', 'off'),
          ('id-d7-2', 'waiting', '/tmp', 'cd-d7-2', 'off'),
          ('id-d7-3', 'waiting', '/tmp', 'cd-d7-3', 'off');"
n=$(claude-director list --limit 1 | jq '.spawns | length')
if [ "$n" != "1" ]; then
    echo "FAIL: expected 1 row, got $n"; exit 1
fi
echo "discover-7 ok: --limit 1 caps result to a single row"
```

# discover-1: list with no filters returns every row

Per SRD §12: an unfiltered `list` enumerates the entire spawns table.

## Setup

Per-case DB-reset. Seed 3 rows directly via sqlite3 — no spawn pipeline
needed for the read-side contract.

## Steps

1. INSERT 3 rows with distinct ids and mixed states/cwds.
2. `agent-director list` → expect 3 rows.

## Pass criteria

- list exits 0.
- `.spawns | length` equals 3.

## Script

```bash
set -euo pipefail
for n in 1 2 3; do
  sqlite3 "$HOME/.agent-director/state.db" \
    "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
     VALUES ('id-d1-$n', 'waiting', '/tmp', 'cd-d1-$n', 'off');"
done
n=$(agent-director list | jq '.spawns | length')
if [ "$n" != "3" ]; then
    echo "FAIL: expected 3 rows, got $n"; exit 1
fi
echo "discover-1 ok: no filters returns all 3 rows"
```

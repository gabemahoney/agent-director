# relay-8: polling timeout → fail-closed deny envelope

Maps to Epic 10 AC #4 (timeout half). With `relay.timeout_seconds=2`
configured and no decide arriving, the polling loop must give up
and emit a deny envelope after ~2 seconds — SRD §6.4 fail-closed
boundary.

## Setup

Per-case DB-reset. Write a per-case `~/.agent-director/config.toml`
pinning `relay.timeout_seconds = 2`.

## Steps

1. Write config.toml with the short timeout.
2. Seed Spawn row.
3. Background-launch hook (it'll poll with the 2s timeout).
4. Do NOT call decide.
5. Wait for hook to exit (~2s).
6. Inspect stdout: deny envelope with default message.
7. Verify elapsed time is between 2 and 5 seconds.

## Pass criteria

- Hook exits 0 in [2s, 5s].
- Stdout is a deny envelope.

## Script

```bash
set -euo pipefail
mkdir -p "$HOME/.agent-director"
printf '[relay]
timeout_seconds = 2
' > "$HOME/.agent-director/config.toml"

sqlite3 "$HOME/.agent-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-r8', 'waiting', '/tmp', 'cd-r8', 'on');"

hook_out=$(mktemp)
start=$(date +%s)
( printf '%s' '{"hook_event_name":"PermissionRequest","tool_name":"Bash"}' \
  | AGENT_DIRECTOR_RELAY_MODE=on \
    AGENT_DIRECTOR_INSTANCE_ID=id-r8 \
    agent-director hook >"$hook_out" 2>/dev/null ) &
hook_pid=$!
wait "$hook_pid"
elapsed=$(( $(date +%s) - start ))

# Reset config so later cases see the default.
rm -f "$HOME/.agent-director/config.toml"

if [ "$elapsed" -lt 2 ] || [ "$elapsed" -gt 5 ]; then
    echo "FAIL: elapsed=${elapsed}s; want 2-5s"; exit 1
fi
behavior=$(jq -r '.hookSpecificOutput.decision.behavior' "$hook_out")
if [ "$behavior" != "deny" ]; then
    echo "FAIL: behavior=$behavior; want deny (fail-closed)"
    cat "$hook_out"; exit 1
fi
echo "relay-8 ok: polling timeout → deny envelope after ${elapsed}s"
```

# release-1-tarball-postinstall-verifies-skill-frontmatter

Pins SR-8 happy path: pack the umbrella against a release-stamped
staging copy, install it into a temp `HOME` with `trustedDependencies`
set, and assert the bundled skill body landed at the expected path
with the staged frontmatter version. This is the same shape the
release pipeline's `verify_phase` step 4/4 runs against the live
release tag — the testplan exercises it standalone so a regression in
files-glob / postinstall-path-resolution / prepack staging surfaces
inside the harness.

## Pass criteria

- `bun pm pack` against the staged umbrella succeeds.
- `bun add` into a fresh consumer project under a temp `HOME` exits 0.
- `${HOME}/.claude/skills/install-agent-director/SKILL.md` exists.
- Its frontmatter `version:` equals the staged release tag.

## Script

```bash
set -euo pipefail

RELEASE_TAG="0.4.1"
STAGE=$(mktemp -d /tmp/rv-r1-stage.XXXXXX)
TESTHOME=$(mktemp -d /tmp/rv-r1-home.XXXXXX)
PROJ=$(mktemp -d /tmp/rv-r1-proj.XXXXXX)

# Stage the umbrella + skill source into a writable copy.
mkdir -p "$STAGE/pkg/ts-bun-client" "$STAGE/skills"
cp -a /work/source/pkg/ts-bun-client/. "$STAGE/pkg/ts-bun-client/"
cp -a /work/source/skills/install-agent-director "$STAGE/skills/"
# Drop any dev artifacts that may have ridden along.
rm -rf "$STAGE/pkg/ts-bun-client/node_modules"
rm -rf "$STAGE/pkg/ts-bun-client/skills"
rm -f "$STAGE/pkg/ts-bun-client/"agent-director-*.tgz

# Stamp the release tag onto both package.json and SKILL.md.
jq --arg v "$RELEASE_TAG" '.version = $v' \
    "$STAGE/pkg/ts-bun-client/package.json" > "$STAGE/pkg/ts-bun-client/package.json.tmp"
mv "$STAGE/pkg/ts-bun-client/package.json.tmp" "$STAGE/pkg/ts-bun-client/package.json"
sed -i "s/^version: .*/version: $RELEASE_TAG/" \
    "$STAGE/skills/install-agent-director/SKILL.md"

# Cross-compile the linux/amd64 CLI binary and stage it into the
# umbrella's platform sub-package so resolveCliPath() can find it.
WORKREPO=$(mktemp -d /tmp/rv-r1-buildrepo.XXXXXX)
cp -a /work/source/. "$WORKREPO/"
(cd "$WORKREPO" && make release-binaries >/dev/null 2>&1)
mkdir -p "$STAGE/pkg/ts-bun-client/platforms/linux-x64/bin"
cp "$WORKREPO/dist/agent-director-linux-amd64" \
    "$STAGE/pkg/ts-bun-client/platforms/linux-x64/bin/agent-director"
chmod 0755 "$STAGE/pkg/ts-bun-client/platforms/linux-x64/bin/agent-director"
rm -rf "$WORKREPO"

# Pack + install + verify.
(cd "$STAGE/pkg/ts-bun-client" \
    && bun install --no-progress >/dev/null 2>&1 \
    && bun run build >/dev/null 2>&1 \
    && bun pm pack >/dev/null 2>&1)
TGZ=$(ls "$STAGE/pkg/ts-bun-client/"agent-director-*.tgz | head -1)
if [ -z "$TGZ" ] || [ ! -f "$TGZ" ]; then
    echo "FAIL: bun pm pack did not produce a tarball under $STAGE"
    ls "$STAGE/pkg/ts-bun-client/" | sed 's/^/  /'
    exit 1
fi

cat > "$PROJ/package.json" <<JSON
{
  "name": "release-verify-r1",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

LOG=$(mktemp /tmp/rv-r1-log.XXXXXX)
EXIT=0
(cd "$PROJ" && HOME="$TESTHOME" bun add "file:$TGZ") > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: bun add exit=$EXIT"
    sed 's/^/  /' "$LOG"
    exit 1
fi

DEST="$TESTHOME/.claude/skills/install-agent-director"
if [ ! -f "$DEST/SKILL.md" ]; then
    echo "FAIL: SKILL.md did not land at $DEST"
    exit 1
fi

LANDED_V=$(awk '/^version:/ {print $2; exit}' "$DEST/SKILL.md")
if [ "$LANDED_V" != "$RELEASE_TAG" ]; then
    echo "FAIL: landed SKILL.md frontmatter version=$LANDED_V; expected $RELEASE_TAG"
    head -6 "$DEST/SKILL.md" | sed 's/^/  /'
    exit 1
fi

rm -rf "$STAGE" "$TESTHOME" "$PROJ" "$LOG"
echo "release-1-tarball-postinstall-verifies-skill-frontmatter ok: tarball install landed SKILL.md frontmatter version=$RELEASE_TAG"
```

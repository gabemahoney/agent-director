# overwrite-2: no --overwrite still errors with ErrTemplateExists

Maps to PRD AC 2. With the new flag landed, the *default* behaviour
(no `--overwrite`) is unchanged from v0.4.2: a second `make-template`
against an existing name exits 1 with `ErrTemplateExists` and does NOT
modify the file. This is the SR-1.6 / SR-8 regression gate.

## Setup

Per-case `HOME=$(mktemp -d)`.

## Steps

1. Run `agent-director make-template --name foo --cwd /tmp/v1` — succeeds.
2. Capture the on-disk body of `foo.toml`.
3. Run `agent-director make-template --name foo --cwd /tmp/v2` (no `--overwrite`) — must exit 1.
4. Assert stderr contains `ErrTemplateExists`.
5. Re-read `foo.toml`; assert content is byte-identical to step 2's snapshot.

## Pass criteria

- First `make-template` exits 0.
- Second `make-template` exits 1.
- Second invocation's stderr contains `ErrTemplateExists`.
- On-disk file is byte-identical to the first call's output.

## Script

```bash
set -euo pipefail
export HOME=$(mktemp -d)
TEMPLATE="$HOME/.agent-director/templates/foo.toml"

/usr/local/bin/agent-director make-template --name foo --cwd /tmp/v1 >/dev/null
before_sha=$(sha256sum "$TEMPLATE" | cut -d' ' -f1)

# Expect non-zero exit. `set +e` so we can capture both the exit code and stderr
# without aborting the script.
set +e
err_output=$(/usr/local/bin/agent-director make-template --name foo --cwd /tmp/v2 2>&1 >/dev/null)
rc=$?
set -e

if [ "$rc" = "0" ]; then
    echo "FAIL: second make-template (no --overwrite) succeeded; want non-zero exit"
    echo "stderr: $err_output"
    exit 1
fi
if [ "$rc" != "1" ]; then
    echo "FAIL: second make-template exit code = $rc; want 1"
    echo "stderr: $err_output"
    exit 1
fi
if ! echo "$err_output" | grep -q "ErrTemplateExists"; then
    echo "FAIL: stderr does not contain ErrTemplateExists"
    echo "stderr: $err_output"
    exit 1
fi

after_sha=$(sha256sum "$TEMPLATE" | cut -d' ' -f1)
if [ "$before_sha" != "$after_sha" ]; then
    echo "FAIL: failed second call modified the file (sha changed from $before_sha to $after_sha)"
    exit 1
fi

echo "overwrite-2 ok: no --overwrite returns ErrTemplateExists exit 1; file unchanged"
```

# Test case: help and --help produce byte-identical stdout

Maps to Epic 1 AC #3.

## Setup

`./bin/agent-director` exists. HOME points at a fresh temp dir.

## Steps

1. Run `HOME=$tmpdir ./bin/agent-director help`; capture stdout into file A.
2. Run `HOME=$tmpdir ./bin/agent-director --help`; capture stdout into file B.
3. Compare A and B byte-for-byte (e.g. `cmp A B`).

## Pass criteria

- Both invocations exit 0.
- `cmp A B` reports no difference (exit 0, no output).

## Notes

The two forms must share the same handler in the CLI dispatcher. A
divergence here usually means someone routed `--help` to a separate stub
without updating the test suite. Useful canary for that kind of regression.

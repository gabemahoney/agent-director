# spawn-5: denied flag in claude_args → ErrSpawnDeniedFlag

Maps to Epic 3 AC #5 (second half): `spawn --cwd /tmp --
--settings={}` → `ErrSpawnDeniedFlag`. The validation step (SRD §7.2
step 3) catches the denied flag before any side effect.

## Setup

Per-case DB-reset has wiped the DB.

## Steps

1. `agent-director spawn --cwd /tmp -- --settings={}` → non-zero exit.
2. Inspect stderr → JSON envelope with `err_name = ErrSpawnDeniedFlag`.
3. Confirm zero spawns rows.

## Pass criteria

- Non-zero exit.
- err_name == ErrSpawnDeniedFlag.
- Zero spawns rows.

## Script

```bash
set -euo pipefail
stderr_file=$(mktemp)
trap 'rm -f "$stderr_file"' EXIT
if agent-director spawn --cwd /tmp -- --settings={} 2>"$stderr_file"; then
    echo "spawn unexpectedly succeeded"; exit 1
fi
err_name="$(jq -r '.err_name' <"$stderr_file")"
test "$err_name" = "ErrSpawnDeniedFlag"
n="$(sqlite3 "$HOME/.agent-director/state.db" 'SELECT COUNT(*) FROM spawns;')"
test "$n" = "0"
echo "spawn-5 ok: ErrSpawnDeniedFlag; no row"
```

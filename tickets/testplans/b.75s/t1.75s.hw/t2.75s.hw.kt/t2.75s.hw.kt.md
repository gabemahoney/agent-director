# env-diff-4-doc-error-prefix-parity

Pins Epic 3 (envelope-diff harness) AC: every documented error case produces
matching `err_name` on both the CLI subprocess path and the in-process Client
path, and the two `err_description` values satisfy the prefix-compatibility
policy (both share the same colon-delimited prefix, or both are identical when
no colon is present). Running `TestEnvelopeDiff_Error` against the live tree
with a clean baseline must pass all subtests.

## Setup

The test harness seeds per-case fixtures internally. The only external
requirement is a built CLI binary (`buildCLI(t)` compiles it from source at
test start). A writable copy of the source tree is used so the `go test`
invocation can build the binary in the module's own module cache.

## Steps

1. Copy `/work/source` to `WORK=$(mktemp -d /tmp/envdiff4_XXXXXX)`.
2. `cd "$WORK"` and run `go test -run TestEnvelopeDiff_Error ./test/envelope-diff/...`.
3. Capture exit code; assert it is 0.
4. Assert that no `--- FAIL` lines appear in the output.
5. Emit pass marker.

## Pass criteria

- `go test -run TestEnvelopeDiff_Error ./test/envelope-diff/...` exits 0.
- No `--- FAIL` lines appear in the test output.
- Subtest lines of the form `--- PASS: TestEnvelopeDiff_Error/<verb>/<ErrName>` appear,
  confirming all documented error cases are exercised.

## Script

```bash
set -euo pipefail

if ! command -v go >/dev/null 2>&1; then
    echo "FAIL: go not found in PATH"
    exit 1
fi

WORK=$(mktemp -d /tmp/envdiff4_XXXXXX)
trap 'rm -rf "$WORK"' EXIT

cp -a /work/source/. "$WORK/"
cd "$WORK"

OUTPUT=$(go test -v -run TestEnvelopeDiff_Error ./test/envelope-diff/... 2>&1) || {
    echo "FAIL: go test -run TestEnvelopeDiff_Error exited non-zero"
    echo "$OUTPUT"
    exit 1
}

if echo "$OUTPUT" | grep -q '^--- FAIL'; then
    echo "FAIL: one or more TestEnvelopeDiff_Error subtests failed"
    echo "$OUTPUT"
    exit 1
fi

CASE_COUNT=$(echo "$OUTPUT" | grep -c 'PASS: TestEnvelopeDiff_Error/' || true)
echo "env-diff-4-doc-error-prefix-parity ok: all documented error cases passed ($CASE_COUNT subtests)"
```

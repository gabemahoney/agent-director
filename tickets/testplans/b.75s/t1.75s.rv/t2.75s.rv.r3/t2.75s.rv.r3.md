# release-3-prepublish-rejects-os-cpu-drift

Pins SR-3.1 drift guard: the umbrella's `prepublishOnly` script
(`scripts/prepublish-guards.ts`) refuses to publish when the
umbrella's `os` array drifts from the exact pin `["linux","darwin"]`.
Stage a release-stamped copy with every other invariant satisfied,
mutate `os` to include `"win32"`, invoke the guard; assert non-zero
exit and the SR-3.1 drift message names the offending field.

## Pass criteria

- `bun run scripts/prepublish-guards.ts` exits non-zero.
- Stderr contains `os field drift` substring.
- Stderr contains `win32` (the drifted element).

## Script

```bash
set -euo pipefail

STAGE=$(mktemp -d /tmp/rv-r3-stage.XXXXXX)

mkdir -p "$STAGE/pkg/ts-bun-client" "$STAGE/skills"
cp -a /work/source/pkg/ts-bun-client/. "$STAGE/pkg/ts-bun-client/"
cp -a /work/source/skills/install-agent-director "$STAGE/skills/"
rm -rf "$STAGE/pkg/ts-bun-client/node_modules"
rm -rf "$STAGE/pkg/ts-bun-client/skills"

# Baseline: all invariants satisfied at 0.4.1.
jq '.version = "0.4.1"
    | .optionalDependencies["@agent-director/linux-x64"]   = "^0.4.1"
    | .optionalDependencies["@agent-director/darwin-arm64"] = "^0.4.1"' \
    "$STAGE/pkg/ts-bun-client/package.json" > "$STAGE/pkg/ts-bun-client/package.json.tmp"
mv "$STAGE/pkg/ts-bun-client/package.json.tmp" "$STAGE/pkg/ts-bun-client/package.json"
sed -i 's/^version: .*/version: 0.4.1/' "$STAGE/skills/install-agent-director/SKILL.md"

# Drift: add win32 to the os array.
jq '.os = ["linux", "darwin", "win32"]' \
    "$STAGE/pkg/ts-bun-client/package.json" > "$STAGE/pkg/ts-bun-client/package.json.tmp"
mv "$STAGE/pkg/ts-bun-client/package.json.tmp" "$STAGE/pkg/ts-bun-client/package.json"

LOG=$(mktemp /tmp/rv-r3-log.XXXXXX)
EXIT=0
(cd "$STAGE/pkg/ts-bun-client" && bun run scripts/prepublish-guards.ts) > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -eq 0 ]; then
    echo "FAIL: prepublish-guards exited 0 — os/cpu drift was not detected"
    sed 's/^/  /' "$LOG"
    exit 1
fi
for needle in "os field drift" "win32"; do
    if ! grep -qF "$needle" "$LOG"; then
        echo "FAIL: SR-3.1 drift message missing substring: $needle"
        sed 's/^/  /' "$LOG"
        exit 1
    fi
done

rm -rf "$STAGE" "$LOG"
echo "release-3-prepublish-rejects-os-cpu-drift ok: prepublish-guards exit=$EXIT with os-drift message naming win32"
```

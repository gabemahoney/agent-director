# overwrite-4: concurrent --overwrite writers leave exactly one valid body

Maps to PRD edge case "another writer is racing the same name". Two
concurrent `make-template --overwrite` invocations against the same
name with distinct `--cwd` values must linearise such that the on-disk
file body equals exactly one of the two writers' encoded bodies. No
zero-byte file, no half-written body, no partial TOML. 3 iterations
(matches the Go-side `TestMakeTemplate_OverwriteTrue_ConcurrentAtomicity`
loop count).

## Setup

Per-case `HOME=$(mktemp -d)`. Pre-warm the state DB once with a
sequential `agent-director help` invocation so the two concurrent
writers do NOT race on the SQLite store's initial schema-setup
exclusive lock — that's incidental to the template-write atomicity
this case pins.

## Steps

For each of 3 iterations:

1. Remove `~/.agent-director/templates/racer.toml` and any `.racer.toml.tmp.*` siblings (idempotent — file may not exist).
2. Background-launch two `agent-director make-template --name racer --overwrite` invocations with distinct `--cwd` values (`/tmp/A` and `/tmp/B`). Both backgrounded with `&`; capture PIDs.
3. `wait` for both PIDs. Both must exit 0.
4. Read `racer.toml`; assert it contains either `cwd = "/tmp/A"` or `cwd = "/tmp/B"`. A body matching neither (including a zero-byte file or a half-written TOML) is a failure.

## Pass criteria

- Both background writers exit 0 in each iteration.
- `racer.toml` is a complete, non-zero-byte TOML containing exactly one of the two writer cwd values in each iteration.
- No orphan `.racer.toml.tmp.*` sibling remains after each iteration.

## Script

```bash
set -euo pipefail
export HOME=$(mktemp -d)
TEMPLATES_DIR="$HOME/.agent-director/templates"
TEMPLATE="$TEMPLATES_DIR/racer.toml"

# Pre-warm the state.db so the two concurrent writers do NOT race on
# SQLite's initial schema-setup exclusive lock. The single sequential
# `help` invocation creates ~/.agent-director/state.db with WAL mode,
# after which concurrent CLI invocations open the DB cleanly.
/usr/local/bin/agent-director help >/dev/null
mkdir -p "$TEMPLATES_DIR"

for i in 1 2 3; do
    # Pre-clean: remove the target and any tempfile orphans.
    rm -f "$TEMPLATE"
    find "$TEMPLATES_DIR" -maxdepth 1 -name '.racer.toml.tmp.*' -delete 2>/dev/null || true

    /usr/local/bin/agent-director make-template --name racer --overwrite --cwd /tmp/A >/dev/null &
    pidA=$!
    /usr/local/bin/agent-director make-template --name racer --overwrite --cwd /tmp/B >/dev/null &
    pidB=$!

    rcA=0; wait "$pidA" || rcA=$?
    rcB=0; wait "$pidB" || rcB=$?
    if [ "$rcA" != "0" ] || [ "$rcB" != "0" ]; then
        echo "FAIL: iter $i — writer exit codes A=$rcA B=$rcB (want 0/0)"
        exit 1
    fi

    if [ ! -f "$TEMPLATE" ]; then
        echo "FAIL: iter $i — $TEMPLATE missing after concurrent writers"; exit 1
    fi
    size=$(stat -c '%s' "$TEMPLATE")
    if [ "$size" -le 0 ]; then
        echo "FAIL: iter $i — $TEMPLATE is zero-byte (atomicity broken)"; exit 1
    fi

    has_A=0; has_B=0
    grep -q 'cwd = "/tmp/A"' "$TEMPLATE" && has_A=1
    grep -q 'cwd = "/tmp/B"' "$TEMPLATE" && has_B=1
    if [ "$has_A" = "0" ] && [ "$has_B" = "0" ]; then
        echo "FAIL: iter $i — on-disk body matches neither writer (half-written body)"
        cat "$TEMPLATE"
        exit 1
    fi
    if [ "$has_A" = "1" ] && [ "$has_B" = "1" ]; then
        echo "FAIL: iter $i — on-disk body contains BOTH cwd values (interleaved write)"
        cat "$TEMPLATE"
        exit 1
    fi

    orphans=$(find "$TEMPLATES_DIR" -maxdepth 1 -name '.racer.toml.tmp.*' 2>/dev/null | wc -l)
    if [ "$orphans" != "0" ]; then
        echo "FAIL: iter $i — orphan tempfile(s) remain:"
        find "$TEMPLATES_DIR" -maxdepth 1 -name '.racer.toml.tmp.*'
        exit 1
    fi
done

echo "overwrite-4 ok: 3 iterations × 2 concurrent writers — body always matches exactly one input"
```

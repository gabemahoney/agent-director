# cabi-1-build-and-symbols

Pins Epic 2 AC: the C-shared build produces a `.so` whose only exported text
symbols are `ad_`-prefixed. `nm -gD libagent_director.so | grep ' T ' |
grep -v ' ad_'` must produce zero lines — no extraneous symbol names leak into
the public ABI.

## Setup

No DB state needed. A writable copy of `/work/source` is created in a temp
directory; `CGO_ENABLED=1 go build -buildmode=c-shared` is run there. The
built `.so` is also stashed to `/tmp/ad_cabi_test.so` for subsequent cases in
the same test run.

## Steps

1. Assert `gcc` is available in PATH; fail with a clear message if not.
2. Copy `/work/source` to `WORK=$(mktemp -d /tmp/cabi1_XXXXXX)`.
3. `cd "$WORK" && mkdir -p dist && CGO_ENABLED=1 go build -buildmode=c-shared -o dist/libagent_director.so ./pkg/cabi`.
4. Assert `dist/libagent_director.so` exists.
5. Copy the `.so` to `/tmp/ad_cabi_test.so` for reuse by later cases.
6. Run `nm -gD dist/libagent_director.so | grep ' T ' | grep -v ' ad_'` and assert the output has zero lines.
7. Count total T symbols and emit pass marker.

## Pass criteria

- `dist/libagent_director.so` exists and is non-empty after the build.
- `nm -gD` reports zero T symbols lacking the `ad_` prefix.
- Script exits 0; final line matches `cabi-1 ok: ...`.

## Script

```bash
set -euo pipefail

if ! command -v gcc >/dev/null 2>&1; then
    echo "FAIL: gcc not found — add build-essential to test/Dockerfile"
    exit 1
fi

WORK=$(mktemp -d /tmp/cabi1_XXXXXX)
cp -a /work/source/. "$WORK/"
cd "$WORK"
mkdir -p dist
CGO_ENABLED=1 go build -buildmode=c-shared -o dist/libagent_director.so ./pkg/cabi

SO="$WORK/dist/libagent_director.so"
if [[ ! -f "$SO" ]]; then
    echo "FAIL: libagent_director.so not built at $SO"
    exit 1
fi

# Stash for subsequent cases in the same run
cp -f "$SO" /tmp/ad_cabi_test.so 2>/dev/null || true

# Positive assertion: count ad_-prefixed T-symbols (our exports).
# Must be exactly 18: 15 verbs + ad_open + ad_close + ad_free_cstring.
# CGO internals (_cgo_*, _cgoexp_*, x_cgo_*, crosscall*, etc.) are
# machinery — not enumerated; we assert only what we own.
COUNT=$(nm -gD "$SO" | grep -c ' T ad_')
if [[ "$COUNT" -ne 18 ]]; then
    echo "FAIL: expected 18 ad_-prefixed T-symbols, got $COUNT"
    nm -gD "$SO" | grep ' T ad_'
    exit 1
fi
echo "cabi-1 ok: 18 ad_-prefixed T-symbols present"
```

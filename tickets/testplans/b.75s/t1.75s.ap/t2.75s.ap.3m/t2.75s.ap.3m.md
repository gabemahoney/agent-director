# pkg-api-3-mcp-routes-through-client

Pins E1 AC: every MCP-server verb dispatch reaches a `pkg/api.Client`
method. The Epic introduced `LiveDispatcher` (`internal/mcp/dispatch.go`)
which holds `client *api.Client` and routes every MCP tool call through
it. This case asserts the routing is intact for every callable verb.

Approach: static check via grep over `internal/mcp/dispatch.go` (and
sibling non-test sources) for `d.client.<Method>(` call sites, one per
callable verb. The dispatcher uses the field name `client` on the
`LiveDispatcher` struct, so `d.client.X` is the canonical call pattern.

## Setup

No DB state needed. Read-only inspection of `/work/source/internal/mcp/`.

## Steps

1. Iterate the 15 expected verb→method pairs (canonical CallableVerbs() set).
2. For each method, `grep -rE "d\.client\.<Method>\b" internal/mcp/*.go`
   on non-test files. Record misses.
3. Assert `internal/mcp/dispatch.go` imports `pkg/api` (defensive sanity
   — otherwise the `d.client.` identifier may not be `*api.Client`).
4. If misses ≠ ∅, fail and print the list.

## Pass criteria

- Every expected `d.client.<Method>(` call site is found at least once in
  `internal/mcp/` non-test sources.
- `internal/mcp/dispatch.go` (or a sibling non-test file) imports
  `github.com/gabemahoney/agent-director/pkg/api`.
- Script exits 0; final line matches `pkg-api-3-mcp-routes-through-client ok: ...`.

## Script

```bash
set -euo pipefail

MCP_DIR=/work/source/internal/mcp

if [[ ! -d "$MCP_DIR" ]]; then
    echo "FAIL: $MCP_DIR not present (source tree layout drifted?)"
    exit 1
fi

# Defensive sanity: at least one non-test .go file imports pkg/api.
IMPORT_HITS=$(grep -lE '"github\.com/gabemahoney/agent-director/pkg/api"' "$MCP_DIR"/*.go 2>/dev/null | grep -v '_test\.go$' || true)
if [[ -z "$IMPORT_HITS" ]]; then
    echo "FAIL: no non-test file under $MCP_DIR imports github.com/gabemahoney/agent-director/pkg/api"
    exit 1
fi

# Canonical CallableVerbs() ordering — keep in sync with
# pkg/api/manifest/manifest_test.go::TestCallableVerbsOrder.
METHODS=(Spawn Status Get SendKeys ReadPane Kill Decide Resume FindMissing Expire Delete MakeTemplate List Pause Version)

mapfile -t SRC_FILES < <(find "$MCP_DIR" -maxdepth 1 -name '*.go' ! -name '*_test.go')
if [[ "${#SRC_FILES[@]}" -eq 0 ]]; then
    echo "FAIL: no non-test .go sources under $MCP_DIR"
    exit 1
fi

MISSES=()
for method in "${METHODS[@]}"; do
    if ! grep -qE "d\.client\.${method}\b" "${SRC_FILES[@]}"; then
        MISSES+=("$method")
    fi
done

if [[ "${#MISSES[@]}" -ne 0 ]]; then
    echo "FAIL: ${#MISSES[@]} callable verb method(s) have no d.client.<Method>( call site under internal/mcp/:"
    printf '  - %s\n' "${MISSES[@]}" >&2
    exit 1
fi

echo "pkg-api-3-mcp-routes-through-client ok: all ${#METHODS[@]} CallableVerbs() methods invoked via d.client.<Method>( under internal/mcp/"
```

# pkg-api-2-cli-routes-through-client

Pins E1 AC: every CLI verb invocation reaches a `pkg/api.Client` method.
The Epic moved the CLI's verb dispatch off direct `internal/store` /
`internal/tmux` access and onto `pkg/api.Client`, so any regression where
a CLI verb bypasses the facade (e.g. a maintainer wiring a new verb
directly to internal state for "performance") is what this case catches.

Approach: static check via grep. For each callable verb name we expect
the corresponding Go method (e.g. `send-keys` → `SendKeys`) to appear in
a `client.<Method>(` call site under `cmd/agent-director/` non-test
sources. If any expected method is missing, fail with the list of misses.
We additionally assert at least one `pkg/api` import exists under
`cmd/agent-director/` so the test isn't trivially satisfied by an
unrelated `client.` identifier.

## Setup

No DB state needed. Read-only inspection of `/work/source/cmd/agent-director/`.

## Steps

1. Iterate the 15 expected verb→method pairs (canonical CallableVerbs() set).
2. For each method, `grep -rE "client\.<Method>\b" cmd/agent-director/*.go`
   on non-test files. Record misses.
3. Assert at least one `cmd/agent-director/*.go` non-test file imports
   `pkg/api` (defensive sanity).
4. If misses ≠ ∅, fail and print the list.

## Pass criteria

- Every expected `client.<Method>(` call site is found at least once in
  `cmd/agent-director/` non-test sources.
- At least one non-test file under `cmd/agent-director/` imports
  `github.com/gabemahoney/agent-director/pkg/api`.
- Script exits 0; final line matches `pkg-api-2-cli-routes-through-client ok: ...`.

## Script

```bash
set -euo pipefail

CMD_DIR=/work/source/cmd/agent-director

if [[ ! -d "$CMD_DIR" ]]; then
    echo "FAIL: $CMD_DIR not present (source tree layout drifted?)"
    exit 1
fi

# Defensive sanity: at least one non-test .go file imports pkg/api.
IMPORT_HITS=$(grep -lE '"github\.com/gabemahoney/agent-director/pkg/api"' "$CMD_DIR"/*.go 2>/dev/null | grep -v '_test\.go$' || true)
if [[ -z "$IMPORT_HITS" ]]; then
    echo "FAIL: no non-test file under $CMD_DIR imports github.com/gabemahoney/agent-director/pkg/api"
    exit 1
fi

# Canonical CallableVerbs() ordering — keep in sync with
# pkg/api/manifest/manifest_test.go::TestCallableVerbsOrder.
METHODS=(Spawn Status Get SendKeys ReadPane Kill Decide Resume FindMissing Expire Delete MakeTemplate List Pause Version)

# Non-test .go files only. Collect into an explicit list rather than
# relying on grep's --include because the test files would otherwise
# be matched.
mapfile -t SRC_FILES < <(find "$CMD_DIR" -maxdepth 1 -name '*.go' ! -name '*_test.go')
if [[ "${#SRC_FILES[@]}" -eq 0 ]]; then
    echo "FAIL: no non-test .go sources under $CMD_DIR"
    exit 1
fi

MISSES=()
for method in "${METHODS[@]}"; do
    if ! grep -qE "client\.${method}\b" "${SRC_FILES[@]}"; then
        MISSES+=("$method")
    fi
done

if [[ "${#MISSES[@]}" -ne 0 ]]; then
    echo "FAIL: ${#MISSES[@]} callable verb method(s) have no client.<Method>( call site under cmd/agent-director/:"
    printf '  - %s\n' "${MISSES[@]}" >&2
    exit 1
fi

echo "pkg-api-2-cli-routes-through-client ok: all ${#METHODS[@]} CallableVerbs() methods invoked via client.<Method>( under cmd/agent-director/"
```

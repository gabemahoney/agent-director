# go-client-3-runnable-examples

Pins Epic 4 AC: the README's equivalent-code Verb examples compile and
pass — the `Example*` functions under `pkg/api/example*_test.go`
(notably `ExampleClient_*`) are runnable, produce their declared
`// Output:` text, and exit clean. This is the strict regression guard
that ensures the customer-facing README snippets keep working as the
public surface evolves.

## Setup

No DB state needed — example tests build their own temp stores. The
script copies `/work/source` to a writable temp dir because `go test`
needs writable scratch space.

## Steps

1. Assert `go` is available in PATH; fail with a clear message if not.
2. Create a writable workspace under `/tmp/` and `cp -a /work/source/.` into it.
3. `cd <workspace> && go test -v -run Example ./pkg/api/...` and capture
   both exit code and stdout.
4. Assert exit code is 0.
5. Assert at least one `=== RUN   Example` line appears in the output —
   `-run Example` matching nothing would be a silent pass and must fail
   loudly.

## Pass criteria

- `go test -v -run Example ./pkg/api/...` exits 0.
- At least one `=== RUN   Example` line appears in the output.
- No `--- FAIL:` lines from any Example test.
- Script exits 0; final line matches `go-client-3-runnable-examples ok: ...`.

## Script

```bash
set -euo pipefail

if ! command -v go >/dev/null 2>&1; then
    echo "FAIL: go not found in PATH"
    exit 1
fi

WORK=$(mktemp -d /tmp/goclient3_XXXXXX)
cp -a /work/source/. "$WORK/"

cd "$WORK"

OUT_FILE=$(mktemp /tmp/goclient3_out.XXXXXX)
EXIT=0
go test -v -run Example ./pkg/api/... > "$OUT_FILE" 2>&1 || EXIT=$?

if [[ "$EXIT" -ne 0 ]]; then
    echo "FAIL: go test -v -run Example ./pkg/api/... exited with code $EXIT"
    tail -200 "$OUT_FILE" >&2
    exit 1
fi

# Make sure at least one Example actually ran. `-run Example` matching
# nothing exits 0 silently — we want a loud failure in that case so a
# refactor that accidentally removes every Example* function is caught.
EXAMPLE_COUNT=$(grep -cE '^=== RUN   Example' "$OUT_FILE" || true)
if [[ "$EXAMPLE_COUNT" -lt 1 ]]; then
    echo "FAIL: no Example* tests ran — '-run Example' matched nothing under pkg/api/"
    tail -50 "$OUT_FILE" >&2
    exit 1
fi

# Defensive: any --- FAIL line is a regression even if the test process
# returned 0 (shouldn't happen, but cheap to assert).
if grep -E '^--- FAIL:' "$OUT_FILE" >/dev/null; then
    echo "FAIL: one or more Example tests reported --- FAIL:"
    grep -E '^--- FAIL:' "$OUT_FILE" >&2
    exit 1
fi

echo "go-client-3-runnable-examples ok: $EXAMPLE_COUNT Example* test(s) ran green under pkg/api/"
```

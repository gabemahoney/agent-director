# env-diff-3-manifest-coverage-gate

Pins Epic 3 (envelope-diff harness) AC: the `nondeterministic.json` coverage
gate (`make nondet-coverage`) fails when a callable verb key is absent from the
manifest. Deleting the `spawn` key from `test/envelope-diff/nondeterministic.json`
causes `go run ./tools/check-nondet` to exit non-zero and print a message to
stderr naming the missing verb. The file is restored unconditionally via `trap`.

## Setup

A writable copy of `/work/source` is required so `nondeterministic.json` can be
mutated. No DB state needed.

## Steps

1. Copy `/work/source` to `WORK=$(mktemp -d /tmp/envdiff3_XXXXXX)`.
2. Register a `trap` to clean up `$WORK` on exit.
3. Record the original content of `test/envelope-diff/nondeterministic.json`.
4. Use `jq 'del(.spawn)'` to remove the `spawn` key; write the result back.
5. `cd "$WORK"` and run `go run ./tools/check-nondet test/envelope-diff/nondeterministic.json`.
6. Capture exit code; assert it is **non-zero**.
7. Capture stderr; assert it contains the string `spawn`.
8. Emit pass marker.

## Pass criteria

- `go run ./tools/check-nondet` exits **non-zero** after the `spawn` key is deleted.
- The stderr output contains `"spawn"` (the tool names the missing verb).
- The original `nondeterministic.json` content is restored before the script exits.

## Script

```bash
set -euo pipefail

if ! command -v jq >/dev/null 2>&1; then
    echo "FAIL: jq not found — add jq to test/Dockerfile"
    exit 1
fi

WORK=$(mktemp -d /tmp/envdiff3_XXXXXX)
MANIFEST_PATH="$WORK/test/envelope-diff/nondeterministic.json"

cleanup() {
    rm -rf "$WORK"
}
trap cleanup EXIT

cp -a /work/source/. "$WORK/"
cd "$WORK"

# Delete the 'spawn' key from nondeterministic.json (in-place via tmp file).
jq 'del(.spawn)' "$MANIFEST_PATH" > "${MANIFEST_PATH}.tmp"
mv "${MANIFEST_PATH}.tmp" "$MANIFEST_PATH"

# Verify the key is gone.
if jq -e 'has("spawn")' "$MANIFEST_PATH" >/dev/null 2>&1; then
    echo "FAIL: 'spawn' key still present after jq del — jq version issue?"
    exit 1
fi

# Run the coverage gate — must fail and name the missing verb.
GATE_ERR=$(go run ./tools/check-nondet "$MANIFEST_PATH" 2>&1) && {
    echo "FAIL: check-nondet exited 0 after removing 'spawn' from manifest"
    exit 1
}

if ! echo "$GATE_ERR" | grep -q 'spawn'; then
    echo "FAIL: check-nondet output does not mention 'spawn'"
    echo "Output: $GATE_ERR"
    exit 1
fi

echo "env-diff-3-manifest-coverage-gate ok: coverage gate fired and named missing verb 'spawn'"
```

# install-1: fresh install in a clean $HOME

Maps to Epic 12 AC #1. The skill runs to completion on a fresh
container; `claude-director help` works post-install; on-disk modes
match SRD §16.2.

## Setup

Per-case DB-reset.

## Steps

1. Invoke install.sh against a fresh temp $HOME, pointing at the
   container's bundled `/usr/local/bin/claude-director` as source.
2. Verify `~/.claude-director/` is mode 0700.
3. Verify `~/.claude-director/state.db` is mode 0600.
4. Verify the canonical binary path exists and is the expected
   symlink → versioned file.
5. Verify `~/.claude/settings.json` contains both hook entries.

## Pass criteria

- install.sh exits 0.
- All mode checks pass.
- Symlink target ends with `claude-director.<vtag>` under
  `~/.claude-director/bin/`.
- settings.json has SessionStart + SessionEnd(matcher=compact) hooks.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)
HOME="$TESTHOME" /opt/skills/install-claude-director/install.sh \
    --binary /usr/local/bin/claude-director \
    --no-symlink >/dev/null

dir_mode=$(stat -c '%a' "$TESTHOME/.claude-director")
if [ "$dir_mode" != "700" ]; then
    echo "FAIL: install dir mode=$dir_mode; want 700"; exit 1
fi

db_mode=$(stat -c '%a' "$TESTHOME/.claude-director/state.db")
if [ "$db_mode" != "600" ]; then
    echo "FAIL: state.db mode=$db_mode; want 600"; exit 1
fi

canonical="$TESTHOME/.claude-director/bin/claude-director"
test -L "$canonical"
target=$(readlink "$canonical")
case "$target" in
    *claude-director.*) ;;
    *) echo "FAIL: symlink target $target does not match versioned pattern"; exit 1 ;;
esac

ss=$(jq -r '.hooks.SessionStart | length' "$TESTHOME/.claude/settings.json")
se=$(jq -r '.hooks.SessionEnd | length' "$TESTHOME/.claude/settings.json")
if [ "$ss" != "1" ] || [ "$se" != "1" ]; then
    echo "FAIL: SessionStart=$ss SessionEnd=$se (want 1/1)"; exit 1
fi
matcher=$(jq -r '.hooks.SessionEnd[0].matcher' "$TESTHOME/.claude/settings.json")
if [ "$matcher" != "compact" ]; then
    echo "FAIL: SessionEnd matcher=$matcher; want compact"; exit 1
fi
echo "install-1 ok: dir 0700, db 0600, symlink → versioned, hooks injected"
```

# spawn-9: fresh-cwd pre-trust flips hasTrustDialogAccepted before spawn

Maps to bug b.f75 AC #1 + #6 (`spawn --cwd <fresh-dir>` results in
state=`waiting` within ~5s without human-in-the-loop, and
`~/.claude.json` projects.<cwd>.hasTrustDialogAccepted is true after
the spawn).

## Setup

Per-case DB-reset has already wiped `~/.claude-director/state.db`. The
container has `claude-director`, `tmux`, `jq` on PATH.

The case uses a synthetic `~/.claude.json` seeded with an empty
projects map. The harness writes to `$HOME/.claude.json` directly —
no real Claude Code install is required, since pre-trust only cares
about the file's shape.

## Steps

1. Seed `$HOME/.claude.json` with `{"projects":{}}` and pick a fresh
   cwd that has never been Claude-Code-trusted (a `mktemp -d`).
2. Assert that `projects.<cwd>.hasTrustDialogAccepted` is unset
   beforehand.
3. `claude-director spawn --cwd <fresh-dir>` → capture
   `claude_instance_id`.
4. Synthesize a SessionStart hook payload and pipe it into
   `CLAUDE_DIRECTOR_INSTANCE_ID=<id> claude-director hook` (mirrors
   what the spawned Claude would emit — keeps the case API-free).
5. `claude-director status --claude-instance-id <id>` → assert
   state = `waiting`.
6. Re-read `$HOME/.claude.json` and assert
   `projects.<canonical cwd>.hasTrustDialogAccepted == true`.

## Pass criteria

- Step 3 exits 0; result JSON has non-empty `claude_instance_id`.
- Step 4 exits 0 with empty stdout (state-tracking fail-open).
- Step 5 returns `{"state":"waiting"}`.
- Step 6 confirms the trust key was flipped to `true` by the spawn.

## Script

```bash
set -euo pipefail
echo '{"projects":{}}' > "$HOME/.claude.json"
cwd="$(cd "$(mktemp -d)" && pwd -P)"  # canonical (symlink-resolved) path
# Precondition: trust key absent.
pre="$(jq -r --arg cwd "$cwd" '.projects[$cwd].hasTrustDialogAccepted // "absent"' "$HOME/.claude.json")"
test "$pre" = "absent"
spawn_out="$(claude-director spawn --cwd "$cwd")"
id="$(printf '%s' "$spawn_out" | jq -r '.claude_instance_id')"
test -n "$id"
hook_out="$(printf '{"hook_event_name":"SessionStart","transcript_path":"/x/abc-uuid.jsonl"}' \
  | CLAUDE_DIRECTOR_INSTANCE_ID="$id" claude-director hook)"
test -z "$hook_out"
state="$(claude-director status --claude-instance-id "$id" | jq -r '.state')"
test "$state" = "waiting"
post="$(jq -r --arg cwd "$cwd" '.projects[$cwd].hasTrustDialogAccepted' "$HOME/.claude.json")"
test "$post" = "true"
echo "spawn-9 ok: id=$id state=waiting hasTrustDialogAccepted=true"
```

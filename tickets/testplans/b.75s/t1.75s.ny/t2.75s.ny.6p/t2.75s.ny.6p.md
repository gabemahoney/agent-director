# overwrite-1: --overwrite replaces an existing template

Maps to PRD AC 1. The second `agent-director make-template --name foo --overwrite` invocation against a pre-existing template succeeds and the on-disk TOML body reflects the second call's params, not the first's.

## Setup

Per-case `HOME=$(mktemp -d)`. No DB-reset required.

## Steps

1. Run `agent-director make-template --name foo --cwd /tmp/v1` against a fresh `${HOME}` — succeeds, creates `${HOME}/.agent-director/templates/foo.toml`.
2. Capture the on-disk body of `foo.toml`.
3. Run `agent-director make-template --name foo --cwd /tmp/v2 --overwrite` — succeeds.
4. Read `foo.toml` back; assert the `cwd` field reflects `/tmp/v2`, not `/tmp/v1`.

## Pass criteria

- Both `make-template` invocations exit 0.
- Post-second-call `foo.toml` contains `cwd = "/tmp/v2"`.
- No orphan `.foo.toml.tmp.*` sibling remains in the templates dir.

## Script

```bash
set -euo pipefail
export HOME=$(mktemp -d)
TEMPLATES_DIR="$HOME/.agent-director/templates"
TEMPLATE="$TEMPLATES_DIR/foo.toml"

/usr/local/bin/agent-director make-template --name foo --cwd /tmp/v1 >/dev/null
if ! grep -q 'cwd = "/tmp/v1"' "$TEMPLATE"; then
    echo "FAIL: first call did not write cwd=/tmp/v1"; cat "$TEMPLATE"; exit 1
fi

/usr/local/bin/agent-director make-template --name foo --cwd /tmp/v2 --overwrite >/dev/null
if ! grep -q 'cwd = "/tmp/v2"' "$TEMPLATE"; then
    echo "FAIL: overwrite did not replace cwd"; cat "$TEMPLATE"; exit 1
fi
if grep -q 'cwd = "/tmp/v1"' "$TEMPLATE"; then
    echo "FAIL: post-overwrite file still contains the original cwd=/tmp/v1"; cat "$TEMPLATE"; exit 1
fi

# No orphan tempfile remains.
orphans=$(find "$TEMPLATES_DIR" -maxdepth 1 -name '.foo.toml.tmp.*' 2>/dev/null | wc -l)
if [ "$orphans" != "0" ]; then
    echo "FAIL: orphan tempfile(s) remain in $TEMPLATES_DIR:"
    find "$TEMPLATES_DIR" -maxdepth 1 -name '.foo.toml.tmp.*'
    exit 1
fi

echo "overwrite-1 ok: --overwrite replaced existing template; no orphan tempfile"
```

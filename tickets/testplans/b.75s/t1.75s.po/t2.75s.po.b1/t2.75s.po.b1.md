# bootstrap-1-fresh-host-supported

Pins the SR-1 happy path end-to-end: `bun add agent-director` on a clean
linux/x86_64 fixture host populates
`${HOME}/.claude/skills/install-agent-director/`. The bundled skill body's
frontmatter `version:` matches the umbrella `package.json` `version`. This
case exercises the FULL bun lifecycle (postinstall lights up automatically
when the package is in `trustedDependencies`). Output-budget verification
lives in the b2/b3/b4 cases which invoke the postinstall directly so
its stdout/stderr is visible (bun hides lifecycle output by design).

## Setup

`ensure_fixtures` (cached at `/tmp/postinstall-skill-copy-fixtures/`)
builds `libagent_director.so`, stages it into `platforms/linux-x64/`,
stamps the umbrella `package.json` and `SKILL.md` to a fixture version,
and packs a tarball. First call ~30s; subsequent calls instant.

## Pass criteria

- `bun add file:.../agent-director.tgz` exits 0.
- `${HOME}/.claude/skills/install-agent-director/SKILL.md` exists.
- The bundled `install.sh` + `uninstall.sh` also landed at the same path.
- Frontmatter `version:` equals the staged umbrella `package.json`
  `version` byte-for-byte.

## Script

```bash
set -euo pipefail

FIXDIR=/tmp/postinstall-skill-copy-fixtures
ensure_fixtures() {
    if [ -f "$FIXDIR/.ready" ] && [ -f "$FIXDIR/agent-director.tgz" ]; then
        return 0
    fi
    rm -rf "$FIXDIR"
    mkdir -p "$FIXDIR"
    local WORK="$FIXDIR/source"
    cp -a /work/source/. "$WORK/"
    # Build the CGO shared library so the umbrella's optional FFI sub-package
    # has a real binary when consumers resolve it. The Pattern A postinstall
    # itself does NOT depend on the .so (it never imports src/), but a built
    # .so lets the b5 ignore-scripts fallback case smoke-test the umbrella's
    # main entry resolution.
    (cd "$WORK" && make libagent_director >/dev/null 2>&1)
    mkdir -p "$WORK/pkg/ts-bun-client/platforms/linux-x64"
    cp "$WORK/dist/libagent_director.so" \
        "$WORK/pkg/ts-bun-client/platforms/linux-x64/libagent_director.so"
    # Stamp a synthetic fixture version on both files so case-1's
    # "frontmatter == package.json version" assertion passes without coupling
    # to whatever ephemeral version the in-tree umbrella package.json carries.
    local FIX_V="0.4.1-rc.fixture"
    jq --arg v "$FIX_V" '.version = $v' \
        "$WORK/pkg/ts-bun-client/package.json" \
        > "$WORK/pkg/ts-bun-client/package.json.tmp"
    mv "$WORK/pkg/ts-bun-client/package.json.tmp" \
        "$WORK/pkg/ts-bun-client/package.json"
    sed -i "s/^version: .*/version: $FIX_V/" \
        "$WORK/skills/install-agent-director/SKILL.md"
    # Pack the tarball — bun pm pack runs the prepack hook which stages
    # ${REPO_ROOT}/skills/install-agent-director/ into
    # pkg/ts-bun-client/skills/install-agent-director/, picked up by the
    # umbrella's files glob.
    (cd "$WORK/pkg/ts-bun-client" \
        && rm -rf node_modules \
        && bun install --no-progress >/dev/null 2>&1 \
        && bun run build >/dev/null 2>&1 \
        && bun pm pack >/dev/null 2>&1)
    mv "$WORK/pkg/ts-bun-client/"agent-director-*.tgz \
        "$FIXDIR/agent-director.tgz"
    printf '%s\n' "$FIX_V" > "$FIXDIR/fixture-version"
    touch "$FIXDIR/.ready"
}

ensure_fixtures
FIX_V=$(cat "$FIXDIR/fixture-version")

TESTHOME=$(mktemp -d /tmp/po-b1-home.XXXXXX)
PROJ=$(mktemp -d /tmp/po-b1-proj.XXXXXX)

cat > "$PROJ/package.json" <<JSON
{
  "name": "consumer-b1",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

LOG=$(mktemp /tmp/po-b1-log.XXXXXX)
EXIT=0
(cd "$PROJ" && HOME="$TESTHOME" bun add "file:$FIXDIR/agent-director.tgz") \
    > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: bun add exit=$EXIT"; sed 's/^/  /' "$LOG"; exit 1
fi

DEST="$TESTHOME/.claude/skills/install-agent-director"
if [ ! -f "$DEST/SKILL.md" ]; then
    echo "FAIL: SKILL.md not landed at $DEST"; sed 's/^/  /' "$LOG"; exit 1
fi
for f in install.sh uninstall.sh; do
    if [ ! -f "$DEST/$f" ]; then
        echo "FAIL: $DEST/$f missing"; exit 1
    fi
done

DEST_V=$(awk '/^version:/ {print $2; exit}' "$DEST/SKILL.md")
PKG_V=$(jq -r .version "$PROJ/node_modules/agent-director/package.json")

if [ "$DEST_V" != "$PKG_V" ]; then
    echo "FAIL: frontmatter version=$DEST_V; package.json version=$PKG_V"
    exit 1
fi
if [ "$DEST_V" != "$FIX_V" ]; then
    echo "FAIL: frontmatter version=$DEST_V did not match staged fixture=$FIX_V"
    exit 1
fi

rm -rf "$TESTHOME" "$PROJ" "$LOG"
echo "bootstrap-1-fresh-host-supported ok: skill at $DEST; version=$DEST_V (==package.json)"
```

# spawn-11: template claude_args flow through when no per-call args

Maps to bug b.qjk (template claude_args silently wiped when caller passes
nothing per-call after `--`). Regression guard: the CLI-level fix in
`cmd/claude-director/spawn_cmd.go` (only assign `p.ClaudeArgs` when
`len(fs.Args()) > 0`).

## Setup

Per-case DB-reset has already wiped `~/.claude-director/state.db`. The
container has `claude-director`, `tmux`, `sqlite3`, `jq` on PATH.

## Steps

1. Create a template with `claude_args` baked in:
   `claude-director make-template --name spawn-11-tpl --claude-args "--model" --claude-args "opus"`
2. Spawn using that template with NO trailing `-- ...` args:
   `claude-director spawn --cwd /tmp --template spawn-11-tpl` → capture `claude_instance_id`.
3. `claude-director get --claude-instance-id <id>` → assert `claude_args` is `["--model", "opus"]`.

## Pass criteria

- Step 1 exits 0.
- Step 2 exits 0 with non-empty `claude_instance_id`.
- Step 3: `claude_args` equals `["--model", "opus"]` — NOT empty/null.
  The regression (b.qjk) produced `null` / `[]` here.

```bash
set -euo pipefail
claude-director make-template --name spawn-11-tpl --claude-args "--model" --claude-args "opus"
spawn_out="$(claude-director spawn --cwd /tmp --template spawn-11-tpl)"
id="$(printf '%s' "$spawn_out" | jq -r '.claude_instance_id')"
test -n "$id"
get_out="$(claude-director get --claude-instance-id "$id")"
model_arg="$(printf '%s' "$get_out" | jq -r '.claude_args[0]')"
opus_arg="$(printf '%s' "$get_out" | jq -r '.claude_args[1]')"
test "$model_arg" = "--model"
test "$opus_arg" = "opus"
echo "spawn-11 ok: id=$id claude_args=[--model,opus]"
```

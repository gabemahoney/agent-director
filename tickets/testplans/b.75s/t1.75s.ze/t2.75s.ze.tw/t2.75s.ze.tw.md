# tmpl-1: make-template writes a hand-readable TOML file

Maps to Epic 7 AC #1. `make-template` produces a TOML file under
`~/.agent-director/templates/`; the file is plain text and decodes
through the same `LoadTemplate` path `spawn --template` will use.

## Setup

Per-case DB-reset. No template prefix file expected.

## Steps

1. `make-template --name tmpl1 --cwd /tmp --label project=foo --allow 'Bash(npm test)'`.
2. `stat` the resulting file (must exist, mode 0600).
3. Inspect the body: contains `cwd = "/tmp"`, `Bash(npm test)`, and
   the `project = "foo"` label.

## Pass criteria

- make-template exits 0; `.path` matches the expected file.
- File exists at `~/.agent-director/templates/tmpl1.toml` (mode 0600).
- Body is plain text containing the expected scalars + array entries.

## Script

```bash
set -euo pipefail
res=$(agent-director make-template --name tmpl1 --cwd /tmp \
        --label project=foo --allow 'Bash(npm test)')
path=$(printf '%s' "$res" | jq -r '.path')
expected="$HOME/.agent-director/templates/tmpl1.toml"
if [ "$path" != "$expected" ]; then
    echo "FAIL: path=$path; want $expected"; exit 1
fi
test -f "$expected"
mode=$(stat -c '%a' "$expected")
if [ "$mode" != "600" ]; then
    echo "FAIL: file mode=$mode; want 600"; exit 1
fi
body=$(cat "$expected")
for want in 'cwd = "/tmp"' 'Bash(npm test)' 'project = "foo"'; do
    if ! echo "$body" | grep -qF "$want"; then
        echo "FAIL: template missing $want"; echo "$body"; exit 1
    fi
done
echo "tmpl-1 ok: hand-readable TOML at $path"
```

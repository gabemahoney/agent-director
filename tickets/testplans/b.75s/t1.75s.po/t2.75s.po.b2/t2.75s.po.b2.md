# bootstrap-2-identical-noop

Pins SR-1.4 identical branch and SR-1.6 output cap on a silent path: when
the destination tree-hash already matches the bundled source, the
postinstall exits 0 with zero filesystem changes and zero stdout/stderr.

Setup uses `bun add --ignore-scripts` to install the umbrella without
firing the postinstall, then pre-seeds the destination by extracting the
skill body from the tarball. Re-running the postinstall manually
(`bun run node_modules/agent-director/scripts/postinstall.ts`) then hits
the identical branch — and crucially gives us **visible** stdout/stderr so
the output budget can be asserted (bun hides lifecycle output by design,
which is why cases b2–b4 invoke the script directly rather than through
`bun add`).

## Pass criteria

- Postinstall exits 0.
- Destination tree-hash is byte-identical between pre- and post-invocation.
- `SKILL.md` mtime is unchanged.
- Zero lines emitted (combined stdout+stderr), filtering for the
  `agent-director: ` prefix.

## Script

```bash
set -euo pipefail

FIXDIR=/tmp/postinstall-skill-copy-fixtures
if [ ! -f "$FIXDIR/.ready" ]; then
    echo "FAIL: fixture cache not populated; bootstrap-1 must run first"
    exit 1
fi

TESTHOME=$(mktemp -d /tmp/po-b2-home.XXXXXX)
PROJ=$(mktemp -d /tmp/po-b2-proj.XXXXXX)

cat > "$PROJ/package.json" <<JSON
{
  "name": "consumer-b2",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

# Install without firing the postinstall — we will invoke it directly.
(cd "$PROJ" && HOME="$TESTHOME" bun add --ignore-scripts \
    "file:$FIXDIR/agent-director.tgz" >/dev/null 2>&1)

PI="$PROJ/node_modules/agent-director/scripts/postinstall.ts"
SRC="$PROJ/node_modules/agent-director/skills/install-agent-director"
DEST="$TESTHOME/.claude/skills/install-agent-director"

# Pre-seed the destination from the same source the postinstall will read,
# making the identical branch the deterministic outcome of the next run.
mkdir -p "$DEST"
cp "$SRC/SKILL.md" "$SRC/install.sh" "$SRC/uninstall.sh" "$DEST/"
chmod 0755 "$DEST/install.sh" "$DEST/uninstall.sh"

tree_hash() {
    (cd "$1" && find . -type f -print0 | sort -z | xargs -0 sha256sum | sha256sum | awk '{print $1}')
}
HASH_BEFORE=$(tree_hash "$DEST")
MTIME_BEFORE=$(stat --format=%Y "$DEST/SKILL.md")

LOG=$(mktemp /tmp/po-b2-log.XXXXXX)
EXIT=0
sleep 1
HOME="$TESTHOME" bun run "$PI" > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: postinstall exit=$EXIT"; sed 's/^/  /' "$LOG"; exit 1
fi

HASH_AFTER=$(tree_hash "$DEST")
MTIME_AFTER=$(stat --format=%Y "$DEST/SKILL.md")

if [ "$HASH_BEFORE" != "$HASH_AFTER" ]; then
    echo "FAIL: tree hash changed; before=$HASH_BEFORE after=$HASH_AFTER"
    exit 1
fi
if [ "$MTIME_BEFORE" != "$MTIME_AFTER" ]; then
    echo "FAIL: SKILL.md mtime changed; before=$MTIME_BEFORE after=$MTIME_AFTER"
    exit 1
fi

POSTINSTALL_LINES=$(grep -c '^agent-director: ' "$LOG" || true)
if [ "$POSTINSTALL_LINES" -ne 0 ]; then
    echo "FAIL: identical branch emitted $POSTINSTALL_LINES line(s) (want 0)"
    grep '^agent-director: ' "$LOG" | sed 's/^/  /'
    exit 1
fi

rm -rf "$TESTHOME" "$PROJ" "$LOG"
echo "bootstrap-2-identical-noop ok: dest tree-hash + mtime unchanged; 0 postinstall lines"
```

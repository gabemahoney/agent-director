# relay-2: deny decision with explicit reason

Maps to Epic 10 AC #3. `decide --decision=deny --reason "not now"`
emits an envelope whose message field carries "not now" verbatim.

## Setup

Per-case DB-reset. Same shape as relay-1.

## Steps

1. Seed Spawn row, background hook.
2. `decide --decision=deny --reason "not now"`.
3. Parse envelope: behavior=deny, message="not now".

## Pass criteria

- Hook stdout's envelope has behavior=deny and the exact reason text.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.claude-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-r2', 'waiting', '/tmp', 'cd-r2', 'on');"

hook_out=$(mktemp)
( printf '%s' '{"hook_event_name":"PermissionRequest","tool_name":"Bash"}' \
  | CLAUDE_DIRECTOR_RELAY_MODE=on \
    CLAUDE_DIRECTOR_INSTANCE_ID=id-r2 \
    claude-director hook >"$hook_out" 2>/dev/null ) &
hook_pid=$!

sleep 0.5
claude-director decide --claude-instance-id id-r2 --decision deny --reason "not now" >/dev/null

wait "$hook_pid"
behavior=$(jq -r '.hookSpecificOutput.decision.behavior' "$hook_out")
message=$(jq -r '.hookSpecificOutput.decision.message' "$hook_out")

if [ "$behavior" != "deny" ] || [ "$message" != "not now" ]; then
    echo "FAIL: envelope behavior=$behavior message=$message; want deny/'not now'"
    cat "$hook_out"; exit 1
fi
echo "relay-2 ok: deny envelope carries explicit reason"
```

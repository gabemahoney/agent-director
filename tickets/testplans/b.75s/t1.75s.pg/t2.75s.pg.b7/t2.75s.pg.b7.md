# bootstrap-7-installsh-arch-probe

Pins SR-2.2: install.sh's `--binary` architecture probe refuses any
binary whose `file(1)` output does not match the host pair. On the
linux/x86_64 harness, a synthetic Mach-O ARM64 file fails the probe;
install.sh exits `2` with the SR-2.2 message before touching
`${HOME}/.agent-director/` or `~/.local/bin/`.

## Pass criteria

- install.sh exit status is `2`.
- Stderr contains:
  - `install.sh: --binary` substring.
  - `architecture mismatch` substring.
  - `host is Linux/x86_64` substring.
  - `Did you pass the wrong --binary?` substring.
- `${HOME}/.agent-director/` was NOT created.
- No `~/.local/bin/agent-director` symlink was written.

## Script

```bash
set -euo pipefail

if ! command -v file >/dev/null 2>&1; then
    echo "FAIL: file(1) missing inside the harness image; check test/Dockerfile"
    exit 1
fi
if ! command -v python3 >/dev/null 2>&1; then
    echo "FAIL: python3 missing inside the harness image"
    exit 1
fi

TESTHOME=$(mktemp -d /tmp/po-b7-home.XXXXXX)

# Synthesize a Mach-O 64-bit ARM64 magic header so file(1) classifies
# the artifact as a darwin-arm64 binary without us building a real one.
# MH_MAGIC_64=0xfeedfacf, CPU_TYPE_ARM64=0x0100000c.
WRONG=$(mktemp /tmp/po-b7-wrong.XXXXXX)
python3 -c "
import struct, sys
data = struct.pack('<IIIIII', 0xfeedfacf, 0x0100000c, 0, 2, 0, 0)
data += b'\x00' * 32
open(sys.argv[1], 'wb').write(data)
" "$WRONG"
chmod +x "$WRONG"

# Confirm file(1) recognizes it as Mach-O arm64 (sanity).
if ! file -b "$WRONG" | grep -q "Mach-O .* arm64"; then
    echo "FAIL: synthetic file was not classified as Mach-O arm64 by file(1)"
    file -b "$WRONG" | sed 's/^/  /'
    exit 1
fi

LOG=$(mktemp /tmp/po-b7-log.XXXXXX)
EXIT=0
HOME="$TESTHOME" bash /work/source/skills/install-agent-director/install.sh \
    --binary "$WRONG" --no-symlink --no-hooks > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 2 ]; then
    echo "FAIL: expected exit 2, got $EXIT"
    sed 's/^/  /' "$LOG"
    exit 1
fi
for needle in \
    "install.sh: --binary" \
    "architecture mismatch" \
    "host is Linux/x86_64" \
    "Did you pass the wrong --binary?"; do
    if ! grep -qF "$needle" "$LOG"; then
        echo "FAIL: SR-2.2 substring missing: $needle"
        sed 's/^/  /' "$LOG"
        exit 1
    fi
done

if [ -e "$TESTHOME/.agent-director" ]; then
    echo "FAIL: ${TESTHOME}/.agent-director was created despite arch-probe refusal"
    ls -la "$TESTHOME/.agent-director" | sed 's/^/  /'
    exit 1
fi
if [ -L "$TESTHOME/.local/bin/agent-director" ] || [ -f "$TESTHOME/.local/bin/agent-director" ]; then
    echo "FAIL: ${TESTHOME}/.local/bin/agent-director was created despite arch-probe refusal"
    exit 1
fi

rm -rf "$TESTHOME" "$WRONG" "$LOG"
echo "bootstrap-7-installsh-arch-probe ok: wrong-arch (Mach-O arm64) refused at preflight; HOME untouched"
```

# ci-matrix-5-runbook-public-repo-security

Pins Epic 6 AC: "The runbook explicitly addresses the security
implications of attaching a self-hosted runner to a public
repository." The Security (CRITICAL) section must reference both the
operator-action settings phrases the Epic body lists verbatim
(`Fork pull request workflows`, `Allow specified actions and reusable
workflows`) and two of the three required GitHub-docs URLs
(approving-workflow-runs-from-public-forks, about-self-hosted-runners).

## Setup

No DB state. Pure read of the committed runbook file.

## Steps

1. Assert the runbook exists.
2. For each required phrase / URL fragment, grep the runbook.
3. If any is missing, fail with the list so the operator sees
   exactly which security guidance is incomplete.
4. Otherwise emit pass marker.

## Pass criteria

- All four required phrases / URL fragments are present.
- Final line matches `ci-matrix-5 ok: ...`.

## Script

```bash
set -euo pipefail

MD="/work/source/docs/self-hosted-runner-setup.md"
if [[ ! -f "$MD" ]]; then
    echo "FAIL: runbook not found at $MD"
    exit 1
fi

# The pinned phrases are a subset of what the Epic body explicitly
# names as required content in the Security (CRITICAL) section:
#   - the fork-PR setting name as it appears in the GitHub UI
#   - the action-allowlist setting name as it appears in the GitHub UI
#   - the GitHub-docs URL for approving-workflow-runs-from-public-forks
#   - the GitHub-docs URL for about-self-hosted-runners (the residual
#     attack-surface reference the Epic body specifically calls out)
REQUIRED=(
    "Fork pull request workflows"
    "Allow specified actions and reusable workflows"
    "approving-workflow-runs-from-public-forks"
    "about-self-hosted-runners"
)

MISSING=()
for phrase in "${REQUIRED[@]}"; do
    if ! grep -qF "$phrase" "$MD"; then
        MISSING+=("$phrase")
    fi
done

if [[ ${#MISSING[@]} -gt 0 ]]; then
    echo "FAIL: runbook missing required security phrase(s):"
    for p in "${MISSING[@]}"; do
        echo "  - $p"
    done
    exit 1
fi

echo "ci-matrix-5 ok: public-repo security phrases present in runbook"
```

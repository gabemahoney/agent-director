# bootstrap-6-host-pair-refused

Pins SR-3.2: the postinstall's host-pair check refuses
`process.platform=darwin, process.arch=x64` (Intel Mac) and
`process.platform=linux, process.arch=arm64` before any filesystem
operation. Each invocation:

- Exits non-zero (status `1` per SR-3.2; the case asserts non-zero so
  future SR adjustments to the specific code do not break this test).
- Writes the SR-3.2 message to stderr containing the literal substrings
  `agent-director: unsupported host: ` and
  `Supported: linux/x64, darwin/arm64`.
- Leaves `${HOME}/.claude/skills/install-agent-director/` un-created.

The simulation uses `Object.defineProperty` because `process.platform`
and `process.arch` are read-only on modern Bun/Node; plain assignment
silently no-ops.

## Pass criteria

For both `(darwin, x64)` and `(linux, arm64)` simulations:

- Exit status is non-zero.
- Stderr contains `agent-director: unsupported host: ` substring.
- Stderr contains `Supported: linux/x64, darwin/arm64` substring.
- `${HOME}/.claude/skills/install-agent-director/` was not created.

## Script

```bash
set -euo pipefail

POSTINSTALL="/work/source/pkg/ts-bun-client/scripts/postinstall.ts"
if [ ! -f "$POSTINSTALL" ]; then
    echo "FAIL: postinstall.ts not at $POSTINSTALL"
    exit 1
fi

run_spoofed() {
    local plat="$1" arch="$2"
    local TESTHOME
    TESTHOME=$(mktemp -d "/tmp/po-b6-$plat-$arch.XXXXXX")
    local LOG
    LOG=$(mktemp /tmp/po-b6-log.XXXXXX)
    local EXIT=0

    HOME="$TESTHOME" bun -e "
        Object.defineProperty(process, 'platform', { value: '$plat' });
        Object.defineProperty(process, 'arch',     { value: '$arch' });
        await import('$POSTINSTALL');
    " > "$LOG" 2>&1 || EXIT=$?

    if [ "$EXIT" -eq 0 ]; then
        echo "FAIL: $plat/$arch — expected non-zero exit, got 0"
        sed 's/^/  /' "$LOG"
        rm -rf "$TESTHOME" "$LOG"
        return 1
    fi
    if ! grep -q "agent-director: unsupported host: " "$LOG"; then
        echo "FAIL: $plat/$arch — SR-3.2 message 'agent-director: unsupported host: ' not in stderr"
        sed 's/^/  /' "$LOG"
        rm -rf "$TESTHOME" "$LOG"
        return 1
    fi
    if ! grep -q "Supported: linux/x64, darwin/arm64" "$LOG"; then
        echo "FAIL: $plat/$arch — SR-3.2 'Supported: linux/x64, darwin/arm64' not in stderr"
        sed 's/^/  /' "$LOG"
        rm -rf "$TESTHOME" "$LOG"
        return 1
    fi
    if [ -e "$TESTHOME/.claude/skills/install-agent-director" ]; then
        echo "FAIL: $plat/$arch — destination was created before refusal"
        ls -R "$TESTHOME/.claude/skills/" | sed 's/^/  /'
        rm -rf "$TESTHOME" "$LOG"
        return 1
    fi
    rm -rf "$TESTHOME" "$LOG"
    return 0
}

run_spoofed darwin x64
run_spoofed linux  arm64

echo "bootstrap-6-host-pair-refused ok: darwin/x64 and linux/arm64 both refused before any filesystem write"
```

# kill-3: kill on unknown id returns ErrSpawnNotFound

Per SRD §13.1: the only surface error for kill is ErrSpawnNotFound,
emitted when no row matches the supplied claude_instance_id.

## Setup

Per-case DB-reset. Empty spawns table.

## Steps

1. `agent-director kill --claude-instance-id absent` (no row exists).
2. Inspect stderr for the typed error envelope.

## Pass criteria

- kill exits non-zero.
- stderr JSON envelope's `err_name` is `ErrSpawnNotFound`.

## Script

```bash
set -euo pipefail
stderr_file=$(mktemp)
if agent-director kill --claude-instance-id absent 2>"$stderr_file" >/dev/null; then
    echo "FAIL: kill on absent id exited 0"; exit 1
fi
err_name=$(jq -r '.err_name' "$stderr_file")
if [ "$err_name" != "ErrSpawnNotFound" ]; then
    echo "FAIL: err_name=$err_name; want ErrSpawnNotFound"; exit 1
fi
echo "kill-3 ok: unknown id → ErrSpawnNotFound"
```

# relay-1: allow decision round-trip

Maps to Epic 10 AC #2. The full happy-path: spawn with relay_mode=on,
PermissionRequest hook starts polling, orchestrator calls `decide
--decision=allow --reason ok`, hook's stdout carries the allow envelope.

## Setup

Per-case DB-reset. The hook is invoked manually with a synthesized
PermissionRequest payload on stdin and `AGENT_DIRECTOR_RELAY_MODE=on`
in env.

## Steps

1. Seed a Spawn row with relay_mode=on via sqlite3.
2. Background-launch `agent-director hook` with the synthesized
   payload and the relay env vars. Capture stdout to a tempfile.
3. Sleep briefly to let the hook reach its polling state.
4. `decide --claude-instance-id <id> --decision allow --reason "ok"`.
5. Wait for the hook process to exit.
6. Parse its stdout: behavior=allow, message=ok.

## Pass criteria

- Hook exits 0.
- decide exits 0.
- Hook stdout is a JSON envelope with behavior=allow and message=ok.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.agent-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode)
   VALUES ('id-r1', 'waiting', '/tmp', 'cd-r1', 'on');"

hook_out=$(mktemp)
( printf '%s' '{"hook_event_name":"PermissionRequest","tool_name":"Bash","tool_input":{"cmd":"echo"}}' \
  | AGENT_DIRECTOR_RELAY_MODE=on \
    AGENT_DIRECTOR_INSTANCE_ID=id-r1 \
    agent-director hook >"$hook_out" 2>/dev/null ) &
hook_pid=$!

sleep 0.5
agent-director decide --claude-instance-id id-r1 --decision allow --reason "ok" >/dev/null

wait "$hook_pid"
behavior=$(jq -r '.hookSpecificOutput.decision.behavior' "$hook_out")
message=$(jq -r '.hookSpecificOutput.decision.message' "$hook_out")

if [ "$behavior" != "allow" ] || [ "$message" != "ok" ]; then
    echo "FAIL: envelope behavior=$behavior message=$message; want allow/ok"
    cat "$hook_out"; exit 1
fi
echo "relay-1 ok: allow round-trip with reason"
```

# resume-2: resume from missing works the same as from ended

Maps to Epic 9 AC #2. A row marked `missing` by find-missing (because
the operator killed it externally) resumes via the same verb with
the same outcome.

## Setup

Per-case DB-reset.

## Steps

1. `spawn --cwd /tmp` → id, session name.
2. Synth SessionStart hook → claude_session_id populated; state=waiting.
3. Seed JSONL on disk at the resolved path.
4. Externally kill the tmux session.
5. Wait for the claude child to exit (its env disappears from /proc).
6. `find-missing` → row transitions to `missing`.
7. Verify row state is `missing`.
8. `resume` → exits 0.
9. New tmux session exists under the canonical name.

## Pass criteria

- find-missing transitions the row to missing.
- resume succeeds; new tmux session up.

## Script

```bash
set -euo pipefail

# Helper: poll /proc for the env-var marker matching `id`.
proc_has() {
    local id="$1"
    for pid in $(ls /proc 2>/dev/null | grep -E '^[0-9]+$'); do
        if [ -r "/proc/$pid/environ" ] && \
            grep -aqz "AGENT_DIRECTOR_INSTANCE_ID=$id" "/proc/$pid/environ" 2>/dev/null; then
            return 0
        fi
    done
    return 1
}

# Spawn a sentinel survivor too — find-missing's degraded-mode guard
# refuses to write when the probe set is empty, so we need at least
# one live claude visible during the find-missing call.
survivor=$(agent-director spawn --cwd /tmp | jq -r '.claude_instance_id')
test -n "$survivor"

id=$(agent-director spawn --cwd /tmp | jq -r '.claude_instance_id')
test -n "$id"
sess=$(agent-director get --claude-instance-id "$id" | jq -r '.tmux_session_name')

session_id="r2-session-uuid"
printf '%s' "{\"hook_event_name\":\"SessionStart\",\"transcript_path\":\"/x/${session_id}.jsonl\"}" \
    | AGENT_DIRECTOR_INSTANCE_ID="$id" agent-director hook >/dev/null

mkdir -p "$HOME/.claude/projects/-tmp"
echo '{}' > "$HOME/.claude/projects/-tmp/${session_id}.jsonl"

# Wait for both env vars visible (survivor must be in /proc for the
# degraded-mode guard to NOT trip).
deadline=$(( $(date +%s) + 10 ))
while [ "$(date +%s)" -lt "$deadline" ]; do
    if proc_has "$survivor" && proc_has "$id"; then
        break
    fi
    sleep 0.2
done

# External kill on the spawn we'll resume.
tmux kill-session -t "$sess"

# Wait until the killed claude env disappears from /proc.
deadline=$(( $(date +%s) + 10 ))
while [ "$(date +%s)" -lt "$deadline" ]; do
    if ! proc_has "$id"; then break; fi
    sleep 0.2
done

agent-director find-missing >/dev/null

state=$(agent-director get --claude-instance-id "$id" | jq -r '.state')
if [ "$state" != "missing" ]; then
    echo "FAIL: state=$state; want missing"; exit 1
fi

agent-director resume --claude-instance-id "$id" >/dev/null

if ! tmux has-session -t "$sess" 2>/dev/null; then
    echo "FAIL: tmux session $sess not up after resume from missing"; exit 1
fi
echo "resume-2 ok: missing → resume → new tmux session up"
```

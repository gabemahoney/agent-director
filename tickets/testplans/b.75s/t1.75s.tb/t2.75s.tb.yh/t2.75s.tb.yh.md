# ts-bun-2-smoke-all-callable-verbs

Pins Epic 5 AC: `bun test test/smoke/` inside `pkg/ts-bun-client/` exits 0
and every callable verb runs. The smoke suite (T7) has one test file per verb
(15 verbs × ≥1 test each). The test runner uses setup.ts as a preload (via
`bunfig.toml`) which builds `bin/ts-helper`, `test/fake-tmux/tmux`, and
`bin/agent-director` from source via `make`.

The `.so` is built fresh from source inside the container (CGO_ENABLED=1)
and copied to `platforms/linux-x64/` and the `node_modules` link so the FFI
loader can resolve it.

## Setup

The script copies `/work/source` to a writable temp dir, builds
`libagent_director.so`, places it in both `platforms/linux-x64/` and
`node_modules/@CHANGEME-H3/agent-director-linux-x64/` (to satisfy the FFI
path resolver), then runs `bun install` and `bun test test/smoke/`.

## Steps

1. Assert `bun` and `go` are in PATH; fail clearly if not.
2. Create workspace; `cp -a /work/source/.`.
3. Build `dist/libagent_director.so` via `CGO_ENABLED=1 go build -buildmode=c-shared`.
4. Copy `.so` to `platforms/linux-x64/` and `node_modules/@CHANGEME-H3/agent-director-linux-x64/`.
5. `cd pkg/ts-bun-client && bun install --no-progress`.
6. `bun test test/smoke/` — setup.ts preload builds ts-helper, fake-tmux, agent-director.
7. Assert exit code 0.
8. Count passing tests from bun summary; assert ≥ 25.

## Pass criteria

- `bun test test/smoke/` exits 0.
- At least 25 tests pass (15 smoke files × ≥1 test each; most have 2).
- Script exits 0; final line contains `ts-bun-2-smoke-all-callable-verbs ok:`.

## Script

```bash
set -euo pipefail

if ! command -v bun >/dev/null 2>&1; then
    echo "FAIL: bun not found in PATH — check test/Dockerfile BUN_VERSION ARG"
    exit 1
fi
if ! command -v go >/dev/null 2>&1; then
    echo "FAIL: go not found in PATH"
    exit 1
fi

export HOME=/tmp/bun-home-tsbun2
mkdir -p "$HOME"

WORK=$(mktemp -d /tmp/tsbun2_XXXXXX)
cp -a /work/source/. "$WORK/"
cd "$WORK"

# Build the C-shared library inside the container
mkdir -p dist
CGO_ENABLED=1 go build -buildmode=c-shared -o dist/libagent_director.so ./pkg/cabi

# Copy .so to platforms dir and node_modules so the FFI resolver finds it
cp dist/libagent_director.so pkg/ts-bun-client/platforms/linux-x64/libagent_director.so
NM_SO="pkg/ts-bun-client/node_modules/@CHANGEME-H3/agent-director-linux-x64/libagent_director.so"
if [[ -d "$(dirname "$NM_SO")" ]]; then
    cp dist/libagent_director.so "$NM_SO"
fi

# Install bun dependencies
cd "$WORK/pkg/ts-bun-client"
bun install --no-progress 2>/dev/null

# Run the smoke suite (setup.ts preload builds ts-helper, fake-tmux, agent-director)
OUT=$(mktemp /tmp/tsbun2_out.XXXXXX)
EXIT=0
bun test test/smoke/ > "$OUT" 2>&1 || EXIT=$?

if [[ "$EXIT" -ne 0 ]]; then
    echo "FAIL: bun test test/smoke/ exited with code $EXIT"
    tail -200 "$OUT" >&2
    exit 1
fi

# Count passing tests from bun summary (format: " N pass" — leading space in bun output)
PASS_COUNT=$(grep -E '[0-9]+ pass$' "$OUT" | grep -oE '[0-9]+' | tail -1 || echo "0")
if [[ "${PASS_COUNT:-0}" -lt 25 ]]; then
    echo "FAIL: expected >=25 passing smoke tests, got ${PASS_COUNT:-0}"
    tail -20 "$OUT" >&2
    exit 1
fi

echo "ts-bun-2-smoke-all-callable-verbs ok: bun test test/smoke/ passed with ${PASS_COUNT} tests"
```

# cabi-4-malformed-json-error

Pins Epic 2 AC: malformed JSON passed to any `ad_*` verb returns a
well-formed error envelope — no panic escapes the C boundary, the process
stays alive with exit code 0, and the returned bytes parse as valid JSON with
an `err_name` field. `ad_open` is used here because it performs its own
`json.Unmarshal` inline and returns `ErrInternal` on parse failure.

## Setup

No DB state needed. The `.so` is reused from `/tmp/ad_cabi_test.so` if present;
otherwise built fresh. A C probe passes `"{ not json"` (truncated,
syntactically invalid JSON) to `ad_open` and captures the returned bytes.

## Steps

1. Assert `gcc` is available in PATH; fail with a clear message if not.
2. Build `.so` to `/tmp/ad_cabi_test.so` if not already present.
3. Write C probe that `dlopen`s the `.so`, resolves `ad_open` and
   `ad_free_cstring`, calls `ad_open("{ not json")`, prints the JSON result,
   and frees the string.
4. Compile with `gcc /tmp/probe4.c -ldl -o /tmp/probe4`.
5. Run `/tmp/probe4 "$SO"` and capture output; assert exit code is 0.
6. Assert the output parses as valid JSON.
7. Assert the JSON has a non-empty `err_name` field.

## Pass criteria

- `/tmp/probe4` exits 0 (no crash, no panic across the C boundary).
- Output is valid JSON.
- `err_name` field is present and non-empty (value will be `"ErrInternal"`).
- Script exits 0; final line matches `cabi-4 ok: ...`.

## Script

```bash
set -euo pipefail

if ! command -v gcc >/dev/null 2>&1; then
    echo "FAIL: gcc not found — add build-essential to test/Dockerfile"
    exit 1
fi

SO="/tmp/ad_cabi_test.so"
if [[ ! -f "$SO" ]]; then
    WORK=$(mktemp -d /tmp/cabi4_XXXXXX)
    cp -a /work/source/. "$WORK/"
    ( cd "$WORK" && mkdir -p dist && CGO_ENABLED=1 go build -buildmode=c-shared -o "$SO" ./pkg/cabi )
fi

cat > /tmp/probe4.c << 'CEOF'
#include <dlfcn.h>
#include <stdio.h>
#include <stdlib.h>
typedef char *(*ad_fn_t)(const char *);
typedef void (*free_fn_t)(char *);
int main(int argc, char **argv) {
    if (argc < 2) { fputs("usage: probe4 <so_path>\n", stderr); return 1; }
    void *lib = dlopen(argv[1], RTLD_NOW);
    if (!lib) { fprintf(stderr, "dlopen: %s\n", dlerror()); return 1; }
    ad_fn_t fn = (ad_fn_t)dlsym(lib, "ad_open");
    if (!fn) { fprintf(stderr, "dlsym ad_open: %s\n", dlerror()); return 1; }
    free_fn_t freefn = (free_fn_t)dlsym(lib, "ad_free_cstring");
    if (!freefn) { fprintf(stderr, "dlsym ad_free_cstring: %s\n", dlerror()); return 1; }
    /* malformed JSON — must not panic; must return a JSON error envelope */
    char *result = fn("{ not json");
    printf("%s\n", result);
    freefn(result);
    dlclose(lib);
    return 0;
}
CEOF
gcc /tmp/probe4.c -ldl -o /tmp/probe4

OUTPUT=$(/tmp/probe4 "$SO")

if ! printf '%s' "$OUTPUT" | jq . >/dev/null 2>&1; then
    echo "FAIL: output is not valid JSON (possible panic or empty return): $OUTPUT"
    exit 1
fi

ERR_NAME=$(printf '%s' "$OUTPUT" | jq -r '.err_name // empty')
if [[ -z "$ERR_NAME" ]]; then
    echo "FAIL: no err_name in malformed-JSON response: $OUTPUT"
    exit 1
fi

echo "cabi-4 ok: malformed JSON to ad_open → err_name=$ERR_NAME, process survived"
```

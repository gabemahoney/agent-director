# install-1: fresh install in a clean $HOME

Maps to Epic 12 AC #1. The skill runs to completion on a fresh
container; `agent-director help` works post-install; on-disk modes
match SRD §16.2.

Post b.43y the install layout is a regular file at the canonical
path (temp+atomic-mv pattern) — no symlink, no versioned siblings.

## Setup

Per-case DB-reset.

## Steps

1. Invoke install.sh against a fresh temp $HOME, pointing at the
   container's bundled `/usr/local/bin/agent-director` as source.
2. Verify `~/.agent-director/` is mode 0700.
3. Verify `~/.agent-director/state.db` is mode 0600.
4. Verify the canonical binary path is a regular file (not a symlink)
   at mode 0755 with the expected size.
5. Verify no versioned-binary siblings exist (legacy pattern is gone).
6. Verify `~/.claude/settings.json` contains both hook entries.

## Pass criteria

- install.sh exits 0.
- All mode checks pass.
- Canonical path is a regular file, mode 0755, size matches source.
- No `agent-director.v*` or `agent-director.t*` artifacts present.
- settings.json has SessionStart + SessionEnd(matcher=compact) hooks.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)
HOME="$TESTHOME" /opt/skills/install-agent-director/install.sh \
    --binary /usr/local/bin/agent-director \
    --no-symlink >/dev/null

dir_mode=$(stat -c '%a' "$TESTHOME/.agent-director")
if [ "$dir_mode" != "700" ]; then
    echo "FAIL: install dir mode=$dir_mode; want 700"; exit 1
fi

db_mode=$(stat -c '%a' "$TESTHOME/.agent-director/state.db")
if [ "$db_mode" != "600" ]; then
    echo "FAIL: state.db mode=$db_mode; want 600"; exit 1
fi

canonical="$TESTHOME/.agent-director/bin/agent-director"
if [ -L "$canonical" ]; then
    echo "FAIL: canonical path is a symlink (pre-b.43y layout)"; exit 1
fi
if [ ! -f "$canonical" ]; then
    echo "FAIL: canonical path is not a regular file"; exit 1
fi
bin_mode=$(stat -c '%a' "$canonical")
if [ "$bin_mode" != "755" ]; then
    echo "FAIL: canonical binary mode=$bin_mode; want 755"; exit 1
fi
src_size=$(stat -c '%s' /usr/local/bin/agent-director)
dst_size=$(stat -c '%s' "$canonical")
if [ "$src_size" != "$dst_size" ]; then
    echo "FAIL: canonical size=$dst_size; want $src_size (source size)"; exit 1
fi

# No versioned-binary siblings from the old layout.
shopt -s nullglob
legacy=("$TESTHOME/.agent-director/bin/"agent-director.v* \
        "$TESTHOME/.agent-director/bin/"agent-director.t2*)
shopt -u nullglob
if [ "${#legacy[@]}" -gt 0 ]; then
    echo "FAIL: legacy versioned siblings present: ${legacy[*]}"; exit 1
fi

ss=$(jq -r '.hooks.SessionStart | length' "$TESTHOME/.claude/settings.json")
se=$(jq -r '.hooks.SessionEnd | length' "$TESTHOME/.claude/settings.json")
if [ "$ss" != "1" ] || [ "$se" != "1" ]; then
    echo "FAIL: SessionStart=$ss SessionEnd=$se (want 1/1)"; exit 1
fi
matcher=$(jq -r '.hooks.SessionEnd[0].matcher' "$TESTHOME/.claude/settings.json")
if [ "$matcher" != "compact" ]; then
    echo "FAIL: SessionEnd matcher=$matcher; want compact"; exit 1
fi
echo "install-1 ok: dir 0700, db 0600, canonical regular file 0755, hooks injected"
```

# install-9: --no-hooks leaves settings.json byte-identical

Maps to b.u9p Acceptance Criterion #3: `--no-hooks` must produce an
install where `~/.claude/settings.json` is byte-identical to its
pre-install state (no edit, no .bak backup, no merge).

This pins the contract that the `--no-hooks` flag is a true skip —
not a "merge to the same shape" no-op that still rewrites the file
and creates a `.bak` sibling. The whole point of `--no-hooks` is to
let an operator install the binary without agent-director ever
touching `~/.claude/settings.json`.

## Setup

Per-case DB-reset. Pre-seed `~/.claude/settings.json` with a known
sentinel payload so a regression that silently merges-then-rewrites
the same content would still flip the sha256.

## Steps

1. Pre-seed `~/.claude/settings.json` with sentinel JSON (a custom
   user `SessionStart` hook plus a top-level marker field), capture
   its sha256 and mtime.
2. Invoke `install.sh --no-hooks` against that `$HOME`.
3. Confirm `install.sh` exits 0.
4. Confirm post-install summary contains `hooks   : skipped (--no-hooks)`.
5. Confirm `~/.claude/settings.json`'s sha256 is unchanged.
6. Confirm no `~/.claude/settings.json.bak.*` sibling was created.
7. Confirm the binary was still installed (the non-hook side of the
   install ran normally — sanity check that `--no-hooks` only gates
   the settings.json edit, not the rest of the script).

## Pass criteria

- `install.sh` exits 0.
- sha256 of `~/.claude/settings.json` is identical before and after.
- No `~/.claude/settings.json.bak.*` files exist post-install.
- Stdout contains the literal `hooks   : skipped (--no-hooks)` line.
- `~/.agent-director/bin/agent-director` is a regular file, mode 0755.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)

# Pre-seed settings.json with sentinel content. The user-hook entry
# would normally be preserved by the merge path; we want to make sure
# even the preserve-and-rewrite case doesn't happen under --no-hooks.
mkdir -p "$TESTHOME/.claude"
cat > "$TESTHOME/.claude/settings.json" <<'JSON'
{
  "u9p_sentinel": "do-not-touch-under-no-hooks",
  "hooks": {
    "SessionStart": [
      {"hooks": [{"type": "command", "command": "echo user-hook"}]}
    ]
  }
}
JSON

before_sha=$(sha256sum "$TESTHOME/.claude/settings.json" | awk '{print $1}')

stdout_file=$(mktemp)
HOME="$TESTHOME" /opt/skills/install-agent-director/install.sh \
    --binary /usr/local/bin/agent-director \
    --no-symlink \
    --no-hooks >"$stdout_file"

after_sha=$(sha256sum "$TESTHOME/.claude/settings.json" | awk '{print $1}')

if [ "$before_sha" != "$after_sha" ]; then
    echo "FAIL: settings.json changed under --no-hooks"
    echo "  before: $before_sha"
    echo "  after : $after_sha"
    diff <(echo "before") <(echo "after") || true
    cat "$stdout_file"
    exit 1
fi

shopt -s nullglob
baks=("$TESTHOME/.claude/settings.json.bak."*)
shopt -u nullglob
if [ "${#baks[@]}" -gt 0 ]; then
    echo "FAIL: --no-hooks created .bak files: ${baks[*]}"
    exit 1
fi

if ! grep -qF 'hooks   : skipped (--no-hooks)' "$stdout_file"; then
    echo "FAIL: stdout missing 'hooks   : skipped (--no-hooks)' line"
    cat "$stdout_file"
    exit 1
fi

canonical="$TESTHOME/.agent-director/bin/agent-director"
if [ ! -f "$canonical" ]; then
    echo "FAIL: canonical binary missing — --no-hooks should only skip hook injection, not the binary install"
    exit 1
fi
bin_mode=$(stat -c '%a' "$canonical")
if [ "$bin_mode" != "755" ]; then
    echo "FAIL: canonical binary mode=$bin_mode; want 755"
    exit 1
fi

echo "install-9 ok: --no-hooks left settings.json byte-identical, no .bak, binary still installed"
```

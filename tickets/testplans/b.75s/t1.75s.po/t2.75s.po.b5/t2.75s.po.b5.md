# bootstrap-5-ignore-scripts-fallback

Pins SR-1.7: when `bun add --ignore-scripts` is used, the postinstall does
NOT run. The destination is NOT created. The umbrella library itself is
still installed and its main entry resolves under `node_modules/`.

## Pass criteria

- `bun add --ignore-scripts file:.../agent-director.tgz` exits 0.
- `${HOME}/.claude/skills/install-agent-director/` does NOT exist.
- `${PROJ}/node_modules/agent-director/dist/index.js` exists (umbrella main
  entry resolves).
- Zero `agent-director: ` lines in combined stdout+stderr (postinstall did
  not run).

## Script

```bash
set -euo pipefail

FIXDIR=/tmp/postinstall-skill-copy-fixtures
if [ ! -f "$FIXDIR/.ready" ]; then
    echo "FAIL: fixture cache not populated; bootstrap-1 must run first"
    exit 1
fi

TESTHOME=$(mktemp -d /tmp/po-b5-home.XXXXXX)
PROJ=$(mktemp -d /tmp/po-b5-proj.XXXXXX)

# Trust the package so the suppression in this case is exclusively from
# --ignore-scripts, not from bun's untrusted-postinstall gate.
cat > "$PROJ/package.json" <<JSON
{
  "name": "consumer-b5",
  "version": "0.0.0",
  "trustedDependencies": ["agent-director"]
}
JSON

LOG=$(mktemp /tmp/po-b5-log.XXXXXX)
EXIT=0
(cd "$PROJ" && HOME="$TESTHOME" bun add --ignore-scripts "file:$FIXDIR/agent-director.tgz") \
    > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 0 ]; then
    echo "FAIL: bun add --ignore-scripts exit=$EXIT"; sed 's/^/  /' "$LOG"; exit 1
fi

if [ -e "$TESTHOME/.claude/skills/install-agent-director" ]; then
    echo "FAIL: destination was created despite --ignore-scripts"
    ls "$TESTHOME/.claude/skills/" | sed 's/^/  /'
    exit 1
fi

UMBRELLA_MAIN="$PROJ/node_modules/agent-director/dist/index.js"
if [ ! -f "$UMBRELLA_MAIN" ]; then
    echo "FAIL: umbrella main entry not at $UMBRELLA_MAIN"
    exit 1
fi

POSTINSTALL_LINES=$(grep -c '^agent-director: ' "$LOG" || true)
if [ "$POSTINSTALL_LINES" -ne 0 ]; then
    echo "FAIL: postinstall emitted $POSTINSTALL_LINES line(s) under --ignore-scripts (want 0)"
    grep '^agent-director: ' "$LOG" | sed 's/^/  /'
    exit 1
fi

rm -rf "$TESTHOME" "$PROJ" "$LOG"
echo "bootstrap-5-ignore-scripts-fallback ok: skill NOT copied; umbrella main entry resolves; postinstall did not run"
```

# bootstrap-8-installsh-uname-gate

Pins SR-2.1: install.sh's uname OS/CPU gate refuses any host outside
`{Linux/x86_64, Darwin/arm64}`. On the linux/x86_64 harness, a wrapper
script on PATH spoofs `uname -s` / `uname -m` to return
`Linux/aarch64` (a rejected tuple); install.sh exits `2` with the
SR-2.1 message before touching `${HOME}/.agent-director/`.

## Pass criteria

- install.sh exit status is `2`.
- Stderr contains:
  - `install.sh: unsupported host: Linux/aarch64` substring.
  - `Supported: Linux/x86_64, Darwin/arm64` substring.
  - `b.fg3` substring (the durable cross-platform expansion anchor).
- `${HOME}/.agent-director/` was NOT created.

## Script

```bash
set -euo pipefail

TESTHOME=$(mktemp -d /tmp/po-b8-home.XXXXXX)
WRAPPER=$(mktemp -d /tmp/po-b8-wrapper.XXXXXX)

# Stage a uname wrapper that returns rejected tuple for -s / -m and
# delegates other flags to the real uname. Path-prepended so install.sh
# resolves this shim instead of the system binary.
cat > "$WRAPPER/uname" <<'WSH'
#!/bin/bash
case "${1:-}" in
    -s) echo "Linux"; exit 0 ;;
    -m) echo "aarch64"; exit 0 ;;
    *)  exec /usr/bin/uname "$@" ;;
esac
WSH
chmod +x "$WRAPPER/uname"

# Sanity: the wrapper must resolve before the real uname.
if [ "$(PATH=$WRAPPER:$PATH uname -m)" != "aarch64" ]; then
    echo "FAIL: uname wrapper did not take precedence on PATH"
    exit 1
fi

LOG=$(mktemp /tmp/po-b8-log.XXXXXX)
EXIT=0
PATH="$WRAPPER:$PATH" HOME="$TESTHOME" bash /work/source/skills/install-agent-director/install.sh \
    --binary /usr/bin/false --no-symlink --no-hooks > "$LOG" 2>&1 || EXIT=$?

if [ "$EXIT" -ne 2 ]; then
    echo "FAIL: expected exit 2, got $EXIT"
    sed 's/^/  /' "$LOG"
    exit 1
fi
for needle in \
    "install.sh: unsupported host: Linux/aarch64" \
    "Supported: Linux/x86_64, Darwin/arm64" \
    "b.fg3"; do
    if ! grep -qF "$needle" "$LOG"; then
        echo "FAIL: SR-2.1 substring missing: $needle"
        sed 's/^/  /' "$LOG"
        exit 1
    fi
done

if [ -e "$TESTHOME/.agent-director" ]; then
    echo "FAIL: ${TESTHOME}/.agent-director was created despite uname-gate refusal"
    exit 1
fi

rm -rf "$TESTHOME" "$WRAPPER" "$LOG"
echo "bootstrap-8-installsh-uname-gate ok: Linux/aarch64 refused at preflight; HOME untouched"
```

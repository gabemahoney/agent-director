# ci-matrix-1-actionlint-clean

Pins Epic 6 AC: `actionlint .github/workflows/cabi-matrix.yml` exits 0
with no warnings — the workflow must remain syntactically valid
GitHub Actions YAML at every commit on the Epic 6 branch.

## Setup

No DB state needed. `actionlint` v1.6.27 is installed inside the case
via `go install` (the Docker image ships Go and has outbound network
access to proxy.golang.org and the module proxy). The install is
scoped to a case-local `GOBIN` so it does not pollute the tester's
default GOPATH between cases.

## Steps

1. Install `actionlint@v1.6.27` into a case-local `GOBIN`.
2. Run `actionlint /work/source/.github/workflows/cabi-matrix.yml`.
3. Assert exit 0 AND stdout/stderr empty (no warnings).
4. Emit pass marker.

## Pass criteria

- `actionlint` binary installs successfully (exit 0 from `go install`).
- `actionlint .../cabi-matrix.yml` exits 0 and emits no diagnostic
  lines on stdout or stderr.
- Final line matches `ci-matrix-1 ok: ...`.

## Script

```bash
set -euo pipefail

WORK=$(mktemp -d /tmp/cim1_XXXXXX)
export GOBIN="$WORK/.bin"
mkdir -p "$GOBIN"
export PATH="$GOBIN:$PATH"

# Pin the actionlint version — bumping is a deliberate change. The
# same v1.6.27 is referenced in .github/workflows/cabi-matrix.yml's
# header comment and in the other workflows' local-lint instructions.
if ! go install github.com/rhysd/actionlint/cmd/actionlint@v1.6.27 2>"$WORK/install.err"; then
    echo "FAIL: go install actionlint failed:"
    cat "$WORK/install.err"
    exit 1
fi

if [[ ! -x "$GOBIN/actionlint" ]]; then
    echo "FAIL: actionlint binary not produced at $GOBIN/actionlint"
    exit 1
fi

YML="/work/source/.github/workflows/cabi-matrix.yml"
if [[ ! -f "$YML" ]]; then
    echo "FAIL: $YML not found (expected at /work/source/...)"
    exit 1
fi

OUT=$("$GOBIN/actionlint" "$YML" 2>&1) || rc=$?
rc=${rc:-0}

if [[ "$rc" -ne 0 ]]; then
    echo "FAIL: actionlint exited $rc on cabi-matrix.yml:"
    echo "$OUT"
    exit 1
fi

if [[ -n "$OUT" ]]; then
    echo "FAIL: actionlint emitted diagnostic output (expected silent on a clean file):"
    echo "$OUT"
    exit 1
fi

echo "ci-matrix-1 ok: actionlint v1.6.27 clean on cabi-matrix.yml"
```

# tmpl-2: spawn --template applies template defaults

Maps to Epic 7 AC #2. After saving a template with `cwd=/tmp`, a
`spawn --template tmpl2` produces a row whose cwd matches.

## Setup

Per-case DB-reset.

## Steps

1. `make-template --name tmpl2 --cwd /tmp --label project=widget`.
2. `spawn --template tmpl2` → capture id.
3. `get --claude-instance-id <id>` → assert cwd=/tmp and label present.

## Pass criteria

- Both verbs exit 0.
- Row's cwd is the template's cwd.
- Row's labels JSON contains the template's `project=widget`.

## Script

```bash
set -euo pipefail
claude-director make-template --name tmpl2 --cwd /tmp --label project=widget >/dev/null
id=$(claude-director spawn --template tmpl2 | jq -r '.claude_instance_id')
test -n "$id"
row=$(claude-director get --claude-instance-id "$id")
cwd=$(printf '%s' "$row" | jq -r '.cwd')
label=$(printf '%s' "$row" | jq -r '.labels.project // empty')
if [ "$cwd" != "/tmp" ]; then
    echo "FAIL: cwd=$cwd; want /tmp"; exit 1
fi
if [ "$label" != "widget" ]; then
    echo "FAIL: labels.project=$label; want widget"; exit 1
fi
echo "tmpl-2 ok: cwd + labels from template applied"
```

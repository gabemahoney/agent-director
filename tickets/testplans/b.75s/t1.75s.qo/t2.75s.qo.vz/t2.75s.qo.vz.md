# spawn-1: spawn returns pending; SessionStart hook flips to waiting

Maps to Epic 3 AC #1 (`spawn --cwd /tmp` returns id; `status` shows
pending immediately; SessionStart hook bumps to waiting).

## Setup

Per-case DB-reset has already wiped `~/.claude-director/state.db`. The
container has `claude-director`, `tmux`, `sqlite3`, `jq` on PATH.

## Steps

1. `claude-director spawn --cwd /tmp` → capture `claude_instance_id`.
2. `claude-director status --claude-instance-id <id>` → assert state =
   `pending`.
3. Synthesize a SessionStart hook payload and pipe it into
   `CLAUDE_DIRECTOR_INSTANCE_ID=<id> claude-director hook`. Exit 0 with
   empty stdout.
4. `claude-director status --claude-instance-id <id>` → assert state =
   `waiting`.

## Pass criteria

- Step 1 exit 0; result JSON has non-empty `claude_instance_id`.
- Step 2 returns `{"state":"pending"}`.
- Step 3 exits 0 with empty stdout (state-tracking fail-open).
- Step 4 returns `{"state":"waiting"}`.

## Script

```bash
set -euo pipefail
spawn_out="$(claude-director spawn --cwd /tmp)"
id="$(printf '%s' "$spawn_out" | jq -r '.claude_instance_id')"
test -n "$id"
state="$(claude-director status --claude-instance-id "$id" | jq -r '.state')"
test "$state" = "pending"
hook_out="$(printf '{"hook_event_name":"SessionStart","transcript_path":"/x/abc-uuid.jsonl"}' \
  | CLAUDE_DIRECTOR_INSTANCE_ID="$id" claude-director hook)"
test -z "$hook_out"
state="$(claude-director status --claude-instance-id "$id" | jq -r '.state')"
test "$state" = "waiting"
session_id="$(claude-director get --claude-instance-id "$id" | jq -r '.claude_session_id')"
test "$session_id" = "abc-uuid"
echo "spawn-1 ok: id=$id state=waiting session_id=$session_id"
```

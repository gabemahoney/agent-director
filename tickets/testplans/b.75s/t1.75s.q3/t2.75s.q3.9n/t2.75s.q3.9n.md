# release-1: make release-binaries produces 4 binaries

Maps to Epic 13 AC #1. `make release-binaries` cross-compiles all
four supported targets into `./dist/`. This case copies the source
tree into a writable temp dir, runs the target, and asserts the
four files exist and are non-empty.

## Setup

Per-case DB-reset. /work/source is the read-only source mount.

## Steps

1. Copy /work/source into a writable temp dir.
2. cd into it; run `make release-binaries`.
3. Verify all four files exist with non-zero size.

## Pass criteria

- make release-binaries exits 0.
- All four files exist in ./dist/.
- Each file is ≥1 MB (binaries this small would be a bad sign).

## Script

```bash
set -euo pipefail
work=$(mktemp -d)
cp -a /work/source/* "$work/"
cd "$work"

make release-binaries >/dev/null 2>&1

for target in linux/amd64 linux/arm64 darwin/amd64 darwin/arm64; do
    os=${target%/*}; arch=${target#*/}
    out="dist/agent-director-${os}-${arch}"
    if [ ! -f "$out" ]; then
        echo "FAIL: $out missing"; exit 1
    fi
    size=$(stat -c '%s' "$out")
    if [ "$size" -lt 1000000 ]; then
        echo "FAIL: $out is only $size bytes (≥1MB expected)"; exit 1
    fi
done
echo "release-1 ok: 4 binaries built and sized"
```

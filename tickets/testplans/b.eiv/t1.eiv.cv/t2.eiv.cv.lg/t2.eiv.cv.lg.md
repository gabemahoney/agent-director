# subprocess-coverage-3-large-stdout

Per SRD §SR-7.1: subprocess stdout (and stderr) must be drained
concurrently with the process so output well past the kernel
pipe-buffer size (~64 KB on Linux) does not deadlock the writer. The
Client achieves this by `Promise.all([response.text(stdout),
response.text(stderr), proc.exited])` — if either pipe is left
unread the subprocess blocks on its next write and the call hangs.

The installed-tarball Client is exercised against a stub CLI binary
that prints a single JSON envelope of ~200 KB to stdout and exits 0.
The Client must parse the envelope and return it within a few
seconds.

## Setup

- `${HOME}=$(mktemp -d)`.
- `bun install file://...tgz` into `${HOME}/proj`.
- Stage a stub CLI binary that prints a 200 KB JSON envelope.

## Steps

1. `client.version({})`.
2. Assert the result has `version`, `commit`, and a `_pad` field of
   the expected size.

## Pass criteria

- Promise resolves within 10 s (a generous bound; in practice well
  under 1 s).
- `v.version === "0.0.0-stub"`, `typeof v.commit === "string"`,
  `v._pad.length >= 180_000`.

## Script

```bash
set -euo pipefail
HOME_DIR="$HOME"
PROJ="$HOME_DIR/proj"
rm -rf "$PROJ"
mkdir -p "$PROJ"
cd "$PROJ"

. /work/source/tickets/testplans/b.eiv/t1.19d.6c/build-tarball.sh

cat > "$PROJ/package.json" <<'EOF'
{
  "name": "subprocess-coverage-lg-consumer",
  "version": "0.0.0",
  "type": "module"
}
EOF

bun install "file://${TARBALL}" >/dev/null

DEST="$PROJ/node_modules/@agent-director/linux-x64"
mkdir -p "$DEST/bin"
cp "$PLATFORM_PKG_DIR/package.json" "$DEST/package.json"

# Stub CLI: prints a single JSON envelope with a ~200 KB padding field,
# then exits 0. Building the pad up-front and printing in a single write
# guarantees the write is far larger than a 64 KB pipe buffer.
cat > "$DEST/bin/agent-director" <<'STUB_EOF'
#!/usr/bin/env bash
size=200000
pad=$(printf '%*s' "$size" '' | tr ' ' 'x')
printf '{"version":"0.0.0-stub","commit":"stub","_pad":"%s"}\n' "$pad"
STUB_EOF
chmod +x "$DEST/bin/agent-director"

cp /work/source/tickets/testplans/b.eiv/t1.eiv.cv/t2.eiv.cv.lg/driver.mjs "$PROJ/driver.mjs"

out=$(bun run "$PROJ/driver.mjs" 2>&1)
echo "$out"
grep -q "OK large-stdout" <<<"$out" || { echo "FAIL: large-stdout assertion missing"; exit 1; }
echo "subprocess-coverage-3-large-stdout ok"
```

# install-6: default uninstall removes hooks + binary but preserves dir

Per SRD §16.2: uninstall without --purge keeps ~/.agent-director/
so operators can hang on to templates and state.db across reinstalls.

## Setup

Per-case DB-reset.

## Steps

1. install.
2. uninstall (no --purge).
3. Verify no agent-director hooks remain.
4. Verify binary at ~/.agent-director/bin/agent-director is gone.
5. Verify ~/.agent-director/ still exists (state.db preserved).

## Pass criteria

- After uninstall: settings.json contains zero agent-director hook
  entries.
- Canonical binary symlink gone.
- ~/.agent-director/state.db still exists.

## Script

```bash
set -euo pipefail
TESTHOME=$(mktemp -d)

HOME="$TESTHOME" /opt/skills/install-agent-director/install.sh \
    --binary /usr/local/bin/agent-director --no-symlink >/dev/null
HOME="$TESTHOME" /opt/skills/install-agent-director/uninstall.sh >/dev/null

prefix="$TESTHOME/.agent-director/bin/agent-director"
cd_count=$(jq --arg p "$prefix" '
    [.hooks.SessionStart[]?, .hooks.SessionEnd[]?
        | select(any(.hooks[]?; (.command // "") | startswith($p)))]
    | length
' "$TESTHOME/.claude/settings.json")
if [ "$cd_count" != "0" ]; then
    echo "FAIL: $cd_count agent-director hook(s) remain post-uninstall"; exit 1
fi
if [ -e "$TESTHOME/.agent-director/bin/agent-director" ]; then
    echo "FAIL: canonical binary still exists"; exit 1
fi
if [ ! -f "$TESTHOME/.agent-director/state.db" ]; then
    echo "FAIL: state.db missing — default uninstall should preserve it"; exit 1
fi
echo "install-6 ok: uninstall removed hooks + binary; preserved state.db"
```

# subprocess-client-3-no-leak

Per SRD §SR-10.5: each subprocess invocation must be fully reaped
before the next call starts. 1000 sequential
`client.list({state:"check_permission"})` calls against an installed
tarball must leave the agent-director process-count delta at zero.

The Docker container always runs linux/x86_64, so the platform gate
that skips this case on darwin (`process.platform === "linux"`) is a
no-op here; the case runs unconditionally.

## Setup

- `${HOME}=$(mktemp -d)`.
- `bun install file:///work/source/dist/agent-director-*.tgz` into
  `${HOME}/proj`.

## Steps

1. Capture baseline: `pgrep -c agent-director` (count of
   currently-running agent-director processes inside the container —
   typically 0).
2. Run a driver that opens one Client and issues 1000 sequential
   `list({state:"check_permission"})` calls.
3. Brief 50ms settling pause.
4. Re-capture the count.

## Pass criteria

- Driver exits 0.
- Post-loop count == baseline count (strict equality).
- Total runtime < 180 s (allows 180ms/call ceiling on busy CI).

## Script

```bash
set -euo pipefail
HOME_DIR="$HOME"
PROJ="$HOME_DIR/proj"
mkdir -p "$PROJ"
cd "$PROJ"

TARBALL=$(ls /work/source/dist/agent-director-*.tgz 2>/dev/null | head -1)
[ -n "$TARBALL" ] || { echo "FAIL: no tarball"; exit 1; }
bun init -y >/dev/null 2>&1
bun install "file://${TARBALL}" >/dev/null

cp /work/source/tickets/testplans/b.eiv/t1.19d.6c/t2.19d.6c.lk/driver.mjs "$PROJ/driver.mjs"

baseline=$(pgrep -c agent-director || echo 0)
"$HOME_DIR/.bun/bin/bun" run "$PROJ/driver.mjs"
sleep 0.05
after=$(pgrep -c agent-director || echo 0)
if [ "$after" != "$baseline" ]; then
    echo "FAIL: pgrep count baseline=${baseline} after=${after}"; exit 1
fi
echo "subprocess-client-3-no-leak ok (baseline=${baseline}, after=${after})"
```

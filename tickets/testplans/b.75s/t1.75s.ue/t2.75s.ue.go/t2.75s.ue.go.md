# resume-1: happy path — resume from ended

Maps to Epic 9 AC #1 (state cycle: spawn → end → resume → new tmux
session up). Stitches together spawn, synthesized SessionStart/End
hooks (Epic 3 driver pattern), JSONL seeding, and the resume verb.

## Setup

Per-case DB-reset.

## Steps

1. `spawn --cwd /tmp` → capture id and tmux_session_name.
2. Synthesize a SessionStart hook with a transcript_path whose
   basename-minus-extension is a known session uuid; this populates
   `claude_session_id` and transitions state pending→waiting.
3. Compute the JSONL path: `$HOME/.claude/projects/-tmp/<session>.jsonl`.
   Create the directory and touch the file (resume's Stat pre-flight
   needs it).
4. Synthesize a SessionEnd hook → state=ended, ended_at stamped.
5. `tmux kill-session -t <original session name>` so the canonical
   name is free.
6. `claude-director resume --claude-instance-id <id>` → exits 0;
   result envelope contains the same id.
7. `tmux has-session -t <name>` exits 0 (new session up).

## Pass criteria

- spawn + each hook firing + resume all exit 0.
- After resume, a tmux session exists under the canonical name.
- Resume's result envelope echoes the same claude_instance_id.

## Script

```bash
set -euo pipefail
id=$(claude-director spawn --cwd /tmp | jq -r '.claude_instance_id')
test -n "$id"
sess=$(claude-director get --claude-instance-id "$id" | jq -r '.tmux_session_name')
test -n "$sess"

# SessionStart synth: claude_session_id will be `r1-session-uuid`.
session_id="r1-session-uuid"
printf '%s' "{\"hook_event_name\":\"SessionStart\",\"transcript_path\":\"/x/${session_id}.jsonl\"}" \
    | CLAUDE_DIRECTOR_INSTANCE_ID="$id" claude-director hook >/dev/null

# Seed the JSONL on disk where resume will Stat it.
jsonl_dir="$HOME/.claude/projects/-tmp"
mkdir -p "$jsonl_dir"
echo '{}' > "$jsonl_dir/${session_id}.jsonl"

# SessionEnd synth → row goes terminal.
printf '%s' '{"hook_event_name":"SessionEnd"}' \
    | CLAUDE_DIRECTOR_INSTANCE_ID="$id" claude-director hook >/dev/null

state=$(claude-director get --claude-instance-id "$id" | jq -r '.state')
if [ "$state" != "ended" ]; then
    echo "FAIL: precondition state=$state; want ended"; exit 1
fi

# Free the canonical tmux session name.
tmux kill-session -t "$sess" 2>/dev/null || true

# Resume.
out=$(claude-director resume --claude-instance-id "$id")
got_id=$(printf '%s' "$out" | jq -r '.claude_instance_id')
if [ "$got_id" != "$id" ]; then
    echo "FAIL: resume returned id=$got_id; want $id"; exit 1
fi

# New tmux session up under the same name.
if ! tmux has-session -t "$sess" 2>/dev/null; then
    echo "FAIL: tmux session $sess not up after resume"; exit 1
fi
echo "resume-1 ok: ended → resume → new tmux session up"
```

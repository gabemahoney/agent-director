# recov-4: expire --older-than 0d reaps all terminal rows

Maps to Epic 8 AC #2. With a zero-duration window, every
ended/missing row is removed; live rows are untouched.

## Setup

Per-case DB-reset. Seed three rows: one ended, one missing, one live.
ended_at is populated for the terminal rows via the ApplyHookTransition
path's existing CURRENT_TIMESTAMP logic. The live row has no ended_at.

## Steps

1. INSERT three rows via sqlite3.
2. Use `agent-director status` (or a direct UPDATE) to flip the
   terminal rows' states via the hook path — but simpler: directly
   set state + ended_at via raw SQL.
3. `expire --older-than 0d`.
4. Verify the two terminal rows are gone; the live row remains.

## Pass criteria

- expire exits 0.
- `count=2`; `.ids` contains both terminal row IDs.
- Live row is untouched.

## Script

```bash
set -euo pipefail
sqlite3 "$HOME/.agent-director/state.db" \
  "INSERT INTO spawns (claude_instance_id, state, cwd, tmux_session_name, relay_mode, ended_at)
   VALUES ('id-ex-end',  'ended',   '/tmp', 'cd-ex-end',  'off', CURRENT_TIMESTAMP),
          ('id-ex-miss', 'missing', '/tmp', 'cd-ex-miss', 'off', CURRENT_TIMESTAMP),
          ('id-ex-live', 'waiting', '/tmp', 'cd-ex-live', 'off', NULL);"

res=$(agent-director expire --older-than 0d)
count=$(printf '%s' "$res" | jq -r '.count')
if [ "$count" != "2" ]; then
    echo "FAIL: count=$count; want 2 (res=$res)"; exit 1
fi
ids=$(printf '%s' "$res" | jq -r '.ids | sort | join(",")')
if [ "$ids" != "id-ex-end,id-ex-miss" ]; then
    echo "FAIL: ids=$ids; want id-ex-end,id-ex-miss"; exit 1
fi

# Live row remains.
n=$(sqlite3 "$HOME/.agent-director/state.db" \
    "SELECT COUNT(*) FROM spawns WHERE claude_instance_id='id-ex-live';")
if [ "$n" != "1" ]; then
    echo "FAIL: live row gone"; exit 1
fi

# Terminal rows are gone.
n=$(sqlite3 "$HOME/.agent-director/state.db" \
    "SELECT COUNT(*) FROM spawns WHERE claude_instance_id IN ('id-ex-end','id-ex-miss');")
if [ "$n" != "0" ]; then
    echo "FAIL: terminal rows survived expire 0d"; exit 1
fi
echo "recov-4 ok: --older-than 0d reaped 2 terminal rows; live row untouched"
```
